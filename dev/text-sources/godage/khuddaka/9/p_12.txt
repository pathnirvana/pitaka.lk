
විසති නිපාතය 
#අම්බපාලී ථෙරී ගාථා
# “මගේ කෙස් බමරුන්සේ කළුවිය. කෙළවර කැරකී නැවී ගියේය. ඔහු දැන් ජරාව හණ නූල් මෙන් වෙයි. සත්‍යවාදීන්ගේ වචනය එසේමැයි. 
# “සුවඳ කරවන ලද සුවඳ කරඬුවක් වැනිවූ මල් පුරවන ලද මාගේ කේශ සමූහය දැන් ජරාවෙන් එළු ලොම් ගඳ හමයි. සත්‍යවාදී වචනය සේ අනිකක් නොවීය. 
# “මනාව වැවූ උයනක් වැනිවූ කොස්සෙන් හා රන් පනාවෙන් අවුල් හළ අගින් සැරසුණු ඒ කේශ සමූහය ජරාවෙන් තන්හි තන්හි ගැලවී වැටී තුනීවිය. සත්‍යවාදී වචනය එසේමැයි. 
# “ගෙතූ හිසකේ වැටිවලින් හොබවනලද රන්කමින් සරසන ලද එය පෙර ශොභමානය, එය ජරායෙන් ගිලිහී වැටී පෑතැටිවිය. සත්‍යවාදී වචනය සේ විය. 
# “දක්ෂ චිත්‍ර ශිල්පියකු විසින් මනාව කළ රේඛා මෙන් මා බැම සඟල පෙර බැබලුහ. දැන් දිරීමෙන් රැලි එල්බ සිටියි. 
# “ප්‍රභාශ්වරවූ ඉතා සිත්කළු නිල්වන් දික්වූ නේත්‍රයෝ මැණික් මෙන් බැබලුහ. දැන් ඔහු ජරාවෙන් මැඩී නොහොබිති. 
# “සිලුටුවූ උස්වූ මනාව පෙනෙන නාසිකාව තරුණ වියට සුදුසු පරිදි හෙබීය. දැන් ඒ ජරාවෙන් ඇකිලී පවතී. 

# “මනා කොට නිමවන ලද රන් වළලු මෙන් මාගේ කන්පෙති බැබලුහ. දැන් ඔහු රැලියෙන් යුක්තව එල්බෙති. 
# “කොඳ මල් කැකුළු වැනි මගේ යහපත් දන්තයෝ පෙර හෙබූහ. දැන් ජරානුභාවයෙන් සෙලැවී කහවිය. 
# “වනලැහැබ්හි හැසිරෙන කෙවිලියක් මෙන් මිහිරි කොට දෙඩූයෙමි. දැන් ජරාවෙන් ඒ ඒ තැන දත් වැටුනේය. 
# “මාගේ යහපත් ශ්‍රීවය මනාව ඔප දැමූ සියුම් රන් සකක් මෙන් පෙර බබළාමය. දැන් එය ජරාවෙන් බිඳී නැවී සිටිසේය. 
# “පෙර මගේ දෙඅත් වටවූ අගුලු දඬු මෙන් ඉතා බබළයි. ඔහු පැළුණු පළොල් අතු මෙන් ජරාය. 
# “සිළුටු මුදු හා රනින් සැරසූ මගේ හස්තයෝ පෙර මනාව හෙබූහ. ඔහු ජරාවෙන් මුලපලා අල මෙන් වෙයි. 
# “පිරී වටව එකට ගැටී උස්වූ මගේ පියකරු පියයුරු පෙර බබළයි. වතුර නැති පෙරහන්කඩ මෙන් එල්බ සිටියි. 
# “මගේ මනා ශරීරය මනාව පිරිමැදි රන්පුවරුවක් මෙන් හොබියි. සියුම් ඇඟ රැළියෙන් ගැවසීය. 
# “පෙර මගේ යහපත් කලවා යුවළ ඇත් සොඬමෙන් බබළයි. ඔහු ජරාවෙන් මඩනාලද්දාවූ උණ පුරුක් මෙන් වෙති. 
# “සිනිඳු රත්රන් පා වළලු වලින් හෙබියාවූ මගේ බබළන ඒ කෙණ්ඩා යුවළ ජරාවෙන් තල දඬුමෙන් වේ. 
# “පෙර මාගේ පතුල් ඉඹුල් පුළුනින් පිරුණාසේ මනාව පැවැත්තාහ. දැන් ජරාවෙන් පැලී රැලි වැටී තිබෙයි. 
# “මේ ශරීරය එබඳු විය. දැදුරුවිය. බොහෝ දුක්ඛයන්ට වාසස්ථාන විය. ඒ ආලේප මැටි ගැලවී වැටුණු ජරපත් ගෙයක්ය. සත්‍යවාදී වචනය එසේම වෙයි.” 
#
# රොහිණී ථෙරී ගාථා
# “පින්වතිය, නුඹ ශ්‍රමණයයි කියා නිදනෙහිය. ශ්‍රමණයයි කියා පුබුදිනෙහිය. ශ්‍රමණයන් ගැනම වර්ණනා කරන්නෙහිය. නුඹ ශ්‍රමණියක් වන්නෙහිය. 

# “බොහෝ ආහාර පාන ශ්‍රමණයන්ට දෙන්නෙහිය. රෝහිණිය, කවර හෙයකින් නුඹට ඒ ශ්‍රමණයන් ප්‍රිය දැයි දැන් නුඹ විචාරමි. 
# “නොකටයුතු කැමතිවූ, කුසීතවූ, අනුන් දුන් දෙයින් ජීවත්වන බලා බලා සිටින, මිහිරි දෙයම කැමැත්තාවූ ඒ ශ්‍රමණයෝ නුඹට කවර හෙයින් ප්‍රියද? 
# “පියාණෙනි, නුඹ බොහෝ කලකින් ශ්‍රමණයන් ගැන මා අතින් විචාළෙහිය. නුඹට උන්වහන්සේලාගේ නුවණ, සිල්, වෑයම කියන්නෙමි. 
# “කටයුතු කැමැත්තාවූ, අනලස්වූ, ශ්‍රෙෂ්ඨ ක්‍රියාවන් කරන්නාවූ උන්වහන්සේලා ඇල්ම තරහ දුරු කෙරෙත්ද එහෙයින් ශ්‍රමණයින් වහන්සේලා මට ප්‍රියයි. 
# “පිරිසිදු ක්‍රියා කරන්නාවූ තුන් අකුසල් මුල් නසත්. උන්වහන්සේලාගේ සියළු පව් ගෙවිණ. එහෙයින් ශ්‍රමණයින් වහන්සේලා මට ප්‍රියය. 
# “ඔවුන්ගේ කයේ කටයුතු පිරිසුදුය. වචනයේ කටයුතුද එසේය. ඔවුන්ගේ සිතේ කටයුතුද පිරිසිදුය එහෙයින් මට මහණහු ප්‍රියය. 
# “පිරිසුදු සක් මෙන්ද මුතු මෙන්ද, ඇතුළත පිටත දෙක පිරිසුදුය. ශුද්ධ ධර්මයන්ගෙන් සම්පූර්ණය. එහෙයින්, 
# “බොහෝ ඇසූ, ධර්මය දරණ, ධර්මයෙන් ජීවත්වන ආර්යයෝ අර්ථයද ධර්මයද ප්‍රකාශ කෙරෙත්. එහෙයින්
# “බොහෝ ඇසූ ධර්මය දරණ, ධර්මයෙන් ජීවත් වන ආර්යයෝ එකඟ සිත් හා පැමිණ සිටි සිහි ඇත්තෝ වෙති. එහෙයින්

# “අතීතයෙහි සැරයටියක් අතේ ඇත්තේ විය. දැන් මුව වැද්දෙකි. තෘෂ්ණාව හේතුකොට ගෙන රළුවූ ගෝරු මඩින් පරතෙර ඒ මට නොහැකිවිය. 
# “චාපාතොම මා කාමයෙහි අතිශයින් මත්වූවෙකැයි සිතා පුතු නලවයි. එහෙයින් මම චාපාවගේ බැම්ම සිඳ නැවතද පැවිදි වන්නෙමි.

# “නාලා නම් ගමින් නික්ම පැවිදිවීමි. දැන් කවරෙක් මේ නාලාගම වෙසේද, දැහැමින් කල් යවන ශ්‍රමණයන් ස්ත්‍රීහු රූපයෙන් බඳිති.

# “නුඹගේ වචනයෙන් ලෝකෝත්තම උත්තර කෙනෙකු නැති බුදුන් වඳිමි. පැදකුණු කොට නුඹට පින් දෙන්නෙමි. 
# “ඉක්බිති කාලතෙම නෙරඤ්ජරා ගංතෙර බලා ගියේය. හෙතෙම දුකද, දුක ඉපදීමද, දුක ඉක්මීමද දුක ඉක්මීමට යායුතු මගවූ උතුම් අටග මගද යන නිවන් පදය බුදුන් දිටී. 
# “මම උන්වහන්සේගේ සිරිපා වැඳ, පැදකුණුකොට චාපාවට කරුණු කියා සස්නෙහි පැවිදි වීමි. තුන් දැනීම ලබන ලදී. සර්වඥ ශාසනය කරණ ලදී. 
#
# සුන්දරී ථෙරී ගාථා
# “පින්වතිය, නුඹ පෙර මියගිය පුත්‍රයන් කමින් රෑ දාවල් දෙක්හි අතිශයින්ම තැවුනෙහිය. 
# “වාසෙට්ඨි බැමිණිය, අද සිය ගණන් දරුවන් කා දමා කවර කාරණයකින් දැඩිසේ ශෝක නොකෙරෙහිද? 
# “බමුණ පෙර නුඹ විසින්ද මා විසින්ද බොහෝ සිය ගණන් දරුවෝද බොහෝ ඥාති වර්ගයාද කා දමන ලදහ. 
# “ඒ මම ඉපදීමෙන් හා මරණයෙන් බේරීම දැන ශොක නොකරමි. නාඬමි. නැවත නොතැවෙමි. 
# “වාසෙට්ඨිය, නුඹ ඒකාන්තයෙන් මෙබඳු පුදුම වචනයක් කීයෙහි. නුඹ කාගේ ධර්මක් දැන ස්ථිරව මෙසේ කියනෙහිද? 
# “බ්‍රාහ්මණය මේ සර්වඥයන් වහන්සේ මිථිලා නුවර සියළු දුක් දුරු කිරීම පිණිස සත්වයින්ට ධර්ම දෙශනා කරණ සේක. 

# “බ්‍රාහ්මණය, අර්හත්වූ ඔබගේ නිදුක්වූ ධර්මය අසා තේරුම් ගත් චතුරාර්ය සත්‍ය ධර්ම ඇත්තී පුත්‍ර ශොකය දුරු කෙළෙමි. 
# “ඒ මමද මිථිලා නගරය බලා යමි. ඒ භාග්‍යවතුන් වහන්සේ මාද සියළු දුකින් මුදවන්නේ නම් ඉතා යෙහෙක. 
# “බ්‍රාහ්මණ තෙම සම්පූර්ණයෙන් මිදුණාවූ උපධි රහිත වූ සර්වඥයන් වහන්සේ දුටුයේය. දුක ඉක්මවූ ඒ සර්වඥයන් වහන්සේ  ඕහට
# “දුකය, දුක් හටගැන්මය, දුක් නැසීමය, දුක් නැසීමේ මගය, යන දුක් නසන ක්‍රමයවූ චතුරාර්ය සත්‍ය ධර්මය දෙශනා කළසේක. 
# “එහි දක්නා ලද බුදු දහම් ඇතියේ ප්‍රව්‍රජ්ජාව රුචි කෙළේය. සුජාතතෙම තුන් රැයකින් තුන් නුවණ ලැබ රහත් විය. 
# “එව රියැදුර, යව, මේ රිය පාවාදී යව. බැමිණියට නිරෝගී බව කියව. දැන් සුජාත බමුණා පැවිදි විය. රෑ තුනින් තුන් නුවණ ලැබීයයි කියව. ඉන්පසු රියැදුරා රියත් මසු දහසත් ගෙන දැන් සුජාත බ්‍රාහ්මණයා පැවිදි විය. රෑ තුනින් තුන් නුවණ ලැබීයයි බ්‍රාහ්මණියට නිරොගී බව කීය. 
##
# “බ්‍රාහ්මණීය, අශ්ව රථයත් මසු දහසත් නුඹටම වේවා. මමද උතුම් සර්වඥයන් වහන්සේ වෙත පැවිදි වෙමි. 
# “ඇත් ගව අසුන්ද, සිත්කළු මිණි කොඩොල්ද පිරුණු ගෘහ සම්පත්තියද හැර ඔබේ පියා පැවිදි විය. සුන්දරිය, නුඹ මේ සම්පත් අනුභව කරව. නුඹ කුල දායාද ඇත්තීය. 
# “ඇත් ගව අසුන්ද, මැණික් තෝඩුද තවත් මේ ගෙයි සිත්කළු සැපද හැර පුත්‍ර ශොකයෙන් පෙළුනු මපියා පැවිදිවිය. සොහොයුරු ශොකයෙන් මිරිකුනු මමද පැවිදිවෙමි. 
# “සුන්දරිය, නුඹ යමක් පතන්නෙහි නම් නුඹේ අදහස සමෘද්ධ වේවා. පිණ්ඩපාත ආහාරයද, සිඟීමද, පාංශුකූල වස්ත්‍රයද යන මේ ලබමින් පරලොවට පව් නැත්තී වෙව. මෙහෙණි සමූහයා බබුළුවන ආර්යාවෙනි, හික්මෙන්නකවූ මා විසින් දිවැස පිරිසිදු කරණ ලදී. මා විසින් පෙර විසූ තැන් දක්නා ලදී. 
##
# “තුන් නුවණ ලබන ලදී. බුද්ධ වචනය කරණලදී. ආර්යාවෙනි, බුදුන් දක්නට සැවැත් නුවර යනු කැමැත්තෙමි. එය දත මැනව. 
# “බුද්ධොත්තමයා සමීපයෙහි සිංහ නාදය පවත්වන්නෙමි. සුන්දරිය, රන්වන්, රන්බඳු සිවි ඇති, නොහික්මුණවුන් හික්මවන, කොයි කරුණකින්වත් බිය නැති, ශාස්තෘවූ සම්බුදුන් බලව. 
# “කෙලෙසුන්ගෙන් මිදුනාවූ උපධි රහිතවූ, පහවූ රාග ඇති බැඳුම් නැති, කටයුතු කළාවූ, පව් රහිතවූ සමීපයට එන සුන්දරිය බලා වදාළ මැනවි. බරණැසින් නික්ම නුඹ වහන්සේ කරා ආවාවූ, 
# “ශ්‍රාවිකාවූ සුන්දරිතොම, මහාවීරයන් වහන්ස, ඔබවහන්සේගේ ශ්‍රීපාදය වඳියි. බ්‍රාහ්මණයන් වහන්ස, නුඹ වහන්සේ සියල්ල දත්සේක. ලෝකයාට අනුශාසනා කරණ සේක. මම කළයුතු කටයුතු කළ, පව් නැති, ළයේ වැඩුණු මුඛයෙන් උපන් ඔබේ දුවණියක් වෙමි. පින්වතිය, නුදුරින් ආ ඒ නුඹට යහපත් ගමනක් වේවා. 
#
# “දැමුනාවූ, සර්වඥ ශ්‍රී පාදයන් වඳනා කැමති, කෙලෙසුන්ගෙන් වෙන්වූ, බැඳුම් නැති, තීමවූ, පව් නැති ශ්‍රාවිකාවෝ මෙසේම එති.”

# කම්මාරධීතා සුභා ථෙරී ගාථා
# පෙර පිරිසිදුවූ, වස්ත්‍රයෙන් සැරසුනාවූ, බාලවූ මම යම් ධර්මයක් ඇසූයෙම්ද ඒ නොපමාවූ මට චතුරාර්ය සත්‍යාවබෝධ විය. 
# “එතැන් පටන් මම සියළු කාමයන්හි මහත් කලකිරීමට පැමිණියෙමි. සිය කයෙහි භය දැක පැවිදි වීමම කැමති වීමි. 
# “මම වනාහි ඥාති සමූහයාද, දැසි දස් කම්කරුවන්ද පිරුණු සිත්කළුවූ මන වඩන ගම් කුඹුරුද හැර

# “මට මේ සුදුසු නොවෙයි. කිසිත් නැති බවම පතමි. යමෙක් රන් රිදී හැරදමා අවුත් නැවත පිළිගන්නේද, 
# “රන් රිදී ආදිය මාර්ගාවබොධයට හෝ නිර්වාණය දැනීමට කරුණු නොවෙයි. ශාන්තිය පිණිස කරුණු නොවේ. මහණකමට සුදුසු නොවෙයි. මෙය ආර්ය ධනයකුත් නොවෙයි. 
#
# “ලෝභ උපදවන්නාවූ මදය එළවන්නාවූ, මුළා කරවන්නාවූ, කෙළෙස් වඩන, සැක උපදවන්නාවූ පරිශ්‍රම ඇති මෙහි ස්ථිර පැවැත්මක් නැත. 
# “මෙහි ඇලුනාවූ, පමාවූ කිලිටි සිත් ඇති මනුෂ්‍යයෝ ඔවුනොවුන් විරුද්ධවූවාහු කලහ වැඩි කෙරෙත්. 
# “කාමයෙහි ඇලුනවුන්ට වධයද, බැඳීමද, පීඩාද, ධන හානියද, ශොක කිරීම හා හැඬීමද පෙනේ. 
# “මාගේ නෑවූ තොපි කුමක් හෙයින් ඒ මා සතුරන් මෙන් කාමයෙහි යොදවාද? කාමයන්හි බිය දක්නා මා පැවිදි වූවකැයි දනුව. 
# “කාමයෝ අමිත්‍රයහ. වධකයෝය, වෛරීහුය. හුල් ඇනීම්ය. අමු රනින් රත්රනින් පව් නොගෙවෙත්. 
# “මාගේ නෑයෝවූ තොපි සතුරන් මෙන් කුමට ඒ මා කාමයෙහි යොදවාද? හිස මුඩුකළ, සඟල සිවුරු දැරූ මා පැවිද්දෙකැයි දනුව. 
# “පිණ්ඩපාතාහාරයද, පිඬු සිඟීමෙහි යාමද, පැවිද්දන්ට සුදුසු පාංශුකූල වස්ත්‍රද යන මේ මට සුදුසුයි. 
# “මහර්ෂීන් විසින් දිව්‍ය මානුෂික කාමයෝ වමාරණ ලදහ. උන්වහන්සේලා නිර්වාණය නිමිති කොට සම්පූර්ණයෙන් මිදුනාහ. නිශ්චල සැපයට පැමිණියාහ. 
# ගමෙක රැකීමක් නැත්ද එබඳු කාමයන් හා මම එක් නොවෙමි. කාමයෝ වනාහි සතුරුය. වධකරුවෝය. ගිනි කඳන් වැනිය. දුක්ය. 

# “ඒ කාමය වනාහි උවදුරු සහිතය. බිය දෙයි. සිත තළයි කටු සහිතය. ගිජු බවට හේතුය. ඉතා විෂමය. ලාටුවකි. මුළාවේ දොරටුවය. 
# “මෝඩ අන්ධ පෘථග්ජනයෝ යමක් ඉතා ප්‍රිය කෙරෙත්ද, ඒ කාමයෝ මහත් උවදුරු සහිතයහ. සර්ප හිසක් වැනිය. භය සහිතය. 
# “ලෝකයෙහි කාම නමැති මඩෙහි ගැලුනු බොහෝ මූඪයෝ ඉපදීමේද, මරණයේද කෙළවර නොදනිති. 
# “මනුෂ්‍යයෝ වනාහි තමහට බොහෝ රෝග ගෙන දෙන කාමය හේතුකොට ඇති, නරකයට යන මාර්ගයට පිළිපදිත්. 
# “සතුරන් උපදවන්නාවූ, තැවීම කරන්නාවූ, ඉතා කෙලෙසන්නාවූ ලෝකයාගේ ඇම වූ බැඳෙන්නාවූ කාමයේ වනාහි පා බඳනාහ. 
# “කාමයෝ වනාහි උමතු කරවන්නාහ. උස්කොට කියවන්නාහ, සිත පෙළන්නාහ, සත්වයන් කෙලෙසීම සඳහා මරු විසින් අටවනලද පාසයෙකි. 
# “කාමයෝ වනාහි කෙළවරක් නැති දොස් ඇත්තාහ. බොහෝ දුක් ඇත්තාහ. ඝොර විෂයහ. ස්වල්ප රස ඇත්තාහ. කෙලෙස් වඩන්නාහ. කුසල ධර්මය වියලවන්නාහ. 
# “හැමවිටම නිර්වාණයෙහි ඇලුනු සිත් ඇති, ඒ මම මෙසේ සලකා කාම හේතුවෙන් හටගන්නා ඒ ව්‍යසනයට නැවත නොඑන්නෙමි. 
# “කෙලෙස් තරණයකොට කාමයන්ගේ සිසිල කැමතිවන යම ඒ බැඳුම් නැසීමට අප්‍රමාදව වාසය කරන්නෙමි. 
# “යමෙකින් බුද්ධාදී මහර්ෂීහු එතර වූවාහුද ශොක රහිතවූ රාගාදී රජස් නැති, නිර්භයවූ, අවංකවූ ආර්ය අෂ්ටාංගික මාර්ගයවූ ඒ මාර්ගයෙහි යමි. 
# “තෘෂ්ණා නැතිකිරීමට පැමිණ රුකක් මුල ධ්‍යාන කරණ කඹුරු දියණියකවූ සුභා නම් මැය දහමින් හොබනාවූ ශ්‍රද්ධා ඇත්තාවූ උත්පලවර්ණාව විසින් හික්මවන ලද තුන් දැනීම ඇති මරු පරයා සිටි මෝතොම පැවිදිව අදට අට දවසකි. 
#
# “නිදහස්වූ ණය නැත්තාවූ වඩනලද ඉන්ද්‍රියයන් ඇති සියළු බැඳුමින් වෙන්වූ කටයුතු නිමි මෝ පව් නැත්තියකි. 
# “සත්වයින්ට අධිපති සක්දෙව් රජ තව්තිසා වැසිදෙව් පිරිස සමග කඹුරු දියණියවූ ශුභා නම් මෙහෙණිය වෙතට ඍද්ධියෙන් පැමිණ නමස්කාර කරයි.”
