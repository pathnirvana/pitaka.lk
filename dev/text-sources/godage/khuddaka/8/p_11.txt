
ද්වාදසක නිපාතය. 
# සීලව ථෙර ගාථා
# “මේ ලොකයෙහි සීලයට අනුවම හික්මිය යුතුයි. මනාකොට හික්මුනාවූ සිල් ඇති තැනැත්තාට මේ ලෝකයෙහි සියලු සම්පත් පමුණුවයි. 
# “ප්‍රශංසාවත්, වස්තු ලාභයත්, පරලොව සැපයත් යන තුන් ආකාර සැපයක් පතන්නාවූ ප්‍රඥාවන්තයා විසින් සීලයම රැකිය යුතුයි. 
# “සිල් ඇති තැනැත්තේ වනාහි සීලයෙන් හික්මීම නිසා බොහෝ මිත්‍රයන් ලබයි. දුශ්ශීල පුද්ගලයා නරක හැසිරීම නිසා යහපත් මිත්‍රයන්ගෙන් පිරිහෙයි. 
# “දුශ්ශීල මනුෂ්‍යයා නින්දා හා අපකීර්තිය ලබයි. සිල්වත් පුද්ගලයා හැමකල්හි ගුණ වර්ණනා හා ප්‍රශංසා ලබයි. 
# “සීලය කුශල ධර්මයන්ට ප්‍රමුඛ වෙයි. සියලු උත්තරී මනුෂ්‍ය ධර්මයන්ට පැමිණීමද වෙයි. සමථ විදර්ශනා ආදී ධර්මයන්ද උපදනේ වෙයි. ඒ නිසා සියලු ධර්මයන්ට මූලිකව පවත්නා සීලය පිරිසිදු කළ යුතුයි. 
# “කාය දුශ්චරිතාදිය වළකන නිසා සීලසංවරය සීමාවක් බඳුය. එසේම සිත සන්තොෂ කරන්නේය. සියලු බුදුවරයන්ගේ තොටක් (එතරවන ස්ථානයක්) වැනිය. ඒ නිසා සීලයම පිරිසිදු කළයුතුය. 
# “සීලය අසමාන බලයක් වැනිය. සීලය උතුම් ආයුධයක් වැනිය. සීලය උතුම් ආභරණයක් වැනිය. සීලය පුදුමවූ යුද ඇඳුමක් බඳුය. 

# “සීලය මහත් බල ඇති සේතුවක් (ඒදණ්ඩක්) වැනිය. සීලය ඉතා උතුම් සුවඳක් වැනිය. දිශානු දිශාවන්ට පැතිරී යන සුවඳ විලේපනයක් වැනිය. 
# “සීලය අග්‍රවූ බත් මුලක් වැනිය. සීලය උතුම්වූ මාර්ගෝපකරණයක් වැනිය. යම් තැනකට යනු කැමැත්තහුට ශ්‍රෙෂ්ඨ වාහනයක් වැනිය. 
# “සීලයෙන් නොහික්මුණු අඥාන පුද්ගලයා මෙලොවද නින්දා ලබයි. පරලොවද නොසතුටු සිත් ඇත්තෙක්ව වෙසෙයි. හැම තැනම නොසතුටෙන් වෙසෙයි. 
# “සීලයට අනුව හික්මුණු පුද්ගලයා මේ ලොකයෙහි දීම කීර්තිය ලබයි. පරලොව ස්වර්ගසැපයෙන් සතුටුවෙයි. හැම තැනදීම ඒ ධෛර්යසම්පන්න පුද්ගලයා සතුටින් වෙසෙයි. 
# “මෙහි සීලයම අග්‍රයි. ප්‍රඥාවද උතුම් වෙයි. මනුෂ්‍ය හා දිව්‍යලොකයෙහිත් සීල ප්‍රඥා දෙකට ජය ලැබෙයි. 
#
# සුනීත ථෙර ගාථා
# “මම නීච කුලයෙහි උපන්නේ ස්වල්ප ආහාර ඇති දිළින්දෙක් වීමි. පරමල් අස්කරන පහත් කර්මාන්තයකද යෙදී සිටියෙමි. 
# “මිනිසුන් විසින් පිළිකුල් කරන ලද්දෙක්වද, අවමන් කරන ලද්දෙක්වද ඝටනා ලද්දෙක්වද විසුවෙමි. එකල්හිද හිස පහත්කොට බොහෝ ජනයා විසින් වඳිනු ලබන්නාවූ 
# “උතුම්වූ මගධපුරයට පිවිසෙන්නාවූ භික්ෂු සංඝයා විසින් පිරිවරණ ලද්දාවූ බුදුරජාණන් වහන්සේ ඒ අවස්ථාවෙහි දුටුවෙමි. 
# “කද බහා තබා උන්වහන්සේට වැඳීමට පැමිණියෙමි. මට අනුකම්පාකොට පුරුෂොත්තමවූ බුදුරජාණන් වහන්සේ වැඩි සිටිසේක. 
# “ශාස්තෲන් වහන්සේගේ පාද වන්දනාකොට එකත් පසෙක සිටියේ සියලුම සත්වයන්ට උතුම්වූ පැවිද්ද මම ඉල්ලුවෙමි. 

# “ඉක්බිති කාරුණිකවූ සියලු ලෝවැස්සන්ට අනුකම්පාකරන්නාවූ ශාස්තෲන් වහන්සේ මට “භික්ෂුව, මෙහි එවයි’ වදාළ සේක. එයින්ම මට උපසම්පදාවද විය. 
# “ඒ මම තනිවම වනයෙහි තෘෂ්ණාවෙන් තොරව වාසය කරන්නේ මට යම් අවවාදයක් දුන්නේද, ඒ ශාස්තෲන් වහන්සේගේ වචනයට අනුව කටයුතු කෙළෙමි. 
# “රාත්‍රිය පිළිබඳ ප්‍රථම යාමයෙහි පෙරවිසූ කඳ පිළිවෙල සිහිකරන ඤාණය ලදිමි. රාත්‍රිය පිළිබඳ මධ්‍යම යාමයෙහි දිව ඇසද පිරිසිදු කෙළෙමි. 
# “රාත්‍රිය පිළිබඳ අළුයම් කාලයෙහි අවිද්‍යාන්ධකාරයද දුරු කෙළෙමි. රාත්‍රිය ගෙවී හිරු උදාවීම ඇතිවූ කල්හි, 
# “උත්තම පුරුෂය ඔබට නමස්කර වේවා” කියා සක්දෙව් රජද මහා බ්‍රහ්මයාද පැමිණ මට නමස්කාර කළාහුය. 
# “නිදුකාණන් වහන්ස, ඔබවහන්සේගේ යම්බඳු ආශ්‍රවයෝ වෙත් නම්, ඒ කෙළෙස් ක්ෂයවී ගියාහ. ඔබ වහන්සේ දක්ෂිණාර්හයෙක් වූයෙහිය. ඉක්බිති දෙවියන් විසින් පිරිවරණ ලද ශාස්තෲන් වහන්සේ මා දැක සිනා පහළකොට මේ කාරණය වදාළහ. 
# “ඉන්ද්‍රිය සංවරයෙන්ද, සීලයෙන්ද, ප්‍රඥාවෙන්ද, ශ්‍රෙෂ්ඨ හැසිරීමෙන්ද යන මෙකී ගුණයන්ගෙන් බ්‍රාහ්මණයෙක් වෙයි. ඒ නිසා මොහු උත්තම බ්‍රාහ්මණයෙකි.”
