තික නිපාතය 
# අංගාණික භාරද්වාජ ථෙර ගාථා
# “නුවණ නැතිව පිරිසිදුකම බලාපොරොත්තුවෙන් වනයෙහි ගිනි දෙවියා පිදීමි. නිවණට මග නොදන්නේ ශරීරයට දුක්දීම කෙළෙමි. 
# “ඒ නිවන් සැප මම ශරීරයට දුක්නොදී සුවසේ ලබාගතිමි. ධර්මයේ යහපත් බව බලන්න. ත්‍රිවිද්‍යාවන්ට ක්‍රමයෙන් පැමුණුනෙමි. බුදුන්ගේ අවවාදය කරණ ලදී. 
# “ජාති මාත්‍රයෙන් බ්‍රහ්මයයි කියන බ්‍රහ්ම බන්ධුවූයෙමි. දැන් පරමාර්ථ බ්‍රාහ්මණයෙක්වීමි. ත්‍රිවිද්‍යා ඇත්තෙකු වී දැන් පරමාර්ථ වශයෙන් ත්‍රිවිද්‍යා ඇත්තෙක්වීමි. සංඥා මාත්‍රයෙන් නෑවාවූ කෙනෙක් වී දැන් පරමාර්ථයෙන් නෑවෙක්වීමි. දැන් පරමාර්ථවේද දත් සොත්ථි’‘ (බ්‍රාහ්මණයෙක්) වීමි.”
# ආයුෂ්මත් අංගාණික භාරද්වාජ තෙමේ මේ ගාථා කීයේය. 
# පච්චය ථෙර ගාථා
# “ශෛක්ෂවූ (රහත්වීම සම්පූර්ණ නොකළ) රහත් බවට නොපැමිණි පැවිදිවී පස්දිනක් ඇති මම මගේ වාසස්ථානයට ගියේ සිතේ මෙවැනි අදහසක් විය. 
# “මගේ හෘදයෙහි තණ්හා හුල නැති නොවනතුරු නොකන්නෙමි නොබොන්නෙමි. පන්සලෙන් පිට නොවන්නෙමි. පැත්තකින් නොනිදන්නෙමි. 
# “මෙසේ වාසය කරන්නාවූ මගේ තද උත්සාහය බලන්න. ත්‍රිවිද්‍යාවන් ලද්දෙමි. බුදුන්ගේ අවවාදය කරණ ලදී.”
# ආයුෂ්මත් පච්චය තෙර මෙසේ මේ ගාථා කීයේය. 
# බක්කුල ථෙර ගාථා
# “ජරාවට පත්වීමට පෙර යමක් කළයුතුද එය ඔහු පසුව කරන්නට කැමතිවේද, හෙතෙම සැපයෙන් පහවේ. පසුවද තැවෙයි. 

# “යමක් කරන්නේද එය කියන්නේය. යමක් නොකරන්නේද එය නොකියන්නේය. පණ්ඩිතයෝ කියන්නාවූ දෙය නොකරන්නාහු විශෙෂයෙන් දැනගනිත්. 
# “යම්තැනක දුක නැතිවේද, බුදුන් විසින් වදාරණ ලද ශොක නැති, කෙලෙස් නැති, නිර්භය නිවන අතිශයින් සැපය.”
# ආයුෂ්මත් බක්කුල තෙර මෙසේ මේ ගාථා කීයේය. 
# ධනිය ථෙර ගාථා
# “මහණකමට ආශාවෙන් ඉදින් සැපසේ ජීවත්වනු කැමැත්තේ නම් සාංඝික සිවුරුද ගිලන්පසද ආහාරද නොඉක්මවා නොසිතන්නේය. 
# “මහණකමට ආශාවෙන් ඉදින් සැපසේ ජීවත් වන්නේ නම් සර්පයා මීගුලෙහි මෙන් සෙනාසනවල වසන්නේය. 
# මහණකමට ආශාවෙන් ඉදින් සැපසේ ජීවත්වනු කැමැත්තේ නම් අප්‍රමාදවූ එකම ධර්මය වඩමින් ලදදෙයින් සතුටුව වසන්නේය.”
# ආයුෂ්මත් ධනිය තෙර මෙසේ මේ ගාථා කීයේය. 
# මාතංග පුත්‍ර ථෙර ගාථා
# “මේ වේලාව බොහෝ සීතලය. මේ ඉතා උෂ්ණය දැන් ඉතා සවස්යයි මෙසේ හරණලද වැඩ ඇති මානවකයන්ට අවස්ථාව වැරදීයයි. 
# “මේ ලොකයෙහි යම් කෙනෙක් සීතලද උෂ්ණයද තණකොළයකට වඩා ගණන් නොගනීද, පුරුෂ වීර්ය කරණ හෙතෙම සැපයෙන් ඈත්නොවෙයි. 
# “දබ්බනම් තණකොළද, කුශතණද, කටු සහිත ගස්ද, සැවැන්දරාද, මුදුතණ හා බඛුස්තණද, විවේකය වඩන මම ලයින් ඉවසන්නෙමි.”
# ආයුෂ්මත් මාතංග පුත්‍ර තෙර තෙමේ මෙසේ මේ ගාථා කීයේය. 
# ඛුජ්ජසොභිත ථෙර ගාථා
# “පැළලුප් නුවර වැසි විසිතුරු ධර්ම කථා ඇති, බොහෝ උගත් ශ්‍රමණයෝ වෙත්ද, ඔවුන් අතරින් කෙනෙකුවූ දීර්ඝායුෂ ඇති ඛුජ්ජසොභිතවූ ආයුෂ්මත් තෙමේ දොරකඩ සිටියි. 

# “පැළලුප් නුවරවැසි විසිතුරු ධර්ම කථා ඇති බොහෝ උගත් ශ්‍රමණයෝ වෙත්ද, ඔවුන් අතරින් කෙනෙකුවූ වාතයෙහි ගමන් කරන ආයුෂ්මත් තෙමේ දොරකඩ සිටියි. 
# “කෙලෙස් යුද්ධයෙන් ධර්මය පිදීමෙන්, කෙලෙස් පැරදවීමෙන් බ්‍රහ්මචර්ය පුරුදු කිරීමෙන් මෙසේ මේ සැපට පැමිණේ.”
# ආයුෂ්මත් බුජ්ජසොභිත තෙර මෙසේ මේ ගාථා කීයේය. 
# වාරණ ථෙර ගාථා
# “මේ ලොකයෙහි මනුෂ්‍යයන් අතුරෙහි යම් කෙනෙක් සතුන් මරාද මෙලොවින්ද පරලොවින්ද යන දෙලොවින් ඒ මනුෂ්‍යතෙමේ පිරිහෙයි. 
# “මෛත්‍රී සිතින් යම් කෙනෙක් සියලු ප්‍රාණීන්ට අනුකම්පාකරයිද, ඒ කරණකොටගෙන එබඳු මනුෂ්‍යතෙම බොහෝ පින් රැස්කරයි. 
# “සුභාෂිතයෙහිත්, ශ්‍රමණයන් ආශ්‍රය කිරීමෙහිත්, තනිව එක ආසනයෙහි සිටීමේත්, සිත සංසිඳවීමෙත් හික්මෙව්.”
# ආයුෂ්මත් වාරණ තෙර මෙසේ මේ ගාථා කීයේය. 
# පස්සික ථෙර ගාථා
# “ශ්‍රද්ධාවත්වූ ඤාණවන්තවූ ධර්මිෂ්ඨවූ සිල්වත්වූ ඒ එක් තැනැත්තා නමුත් ශ්‍රද්ධාව නැති මේ ඤාතීවූ දන්නාවූ හිතවතුන්ට වඩා යහපත් වේ. 
# “ඥාති බන්ධු ආශාවෙන් භික්ෂූන් අතර ක්‍රියාකොට මා විසින් නිග්‍රහ කිරීමෙන් හා අනුග්‍රහ කිරීමෙන් ඥාතීන්ට චොදනා කරණලදී. 
# “ඉක්මගියාවූ කළුරිය කළ ඔව්හු දිව්‍යසම්පත්තියට පැමිණියාහුය. මගේ සහෝදරවරුද මවුද කැමති දේ ලැබ සතුටුවෙත්.”
# ආයුෂ්මත් පස්සික තෙර මෙසේ මේ ගාථා කීයේය. 
# යසොජ ථෙර ගාථා
# “දන්තිලතා නම් වැලක පුරුකක් වැනි කෙට්ටුවූ නහර මතුවීගිය, ආහාරපානයන්හි පමණ දන්නාවූ කම්මැලි නැති ස්වභාව ඇති පුරුෂ තෙමේ, 
# “කැලේ මහවනයෙහි ඩැහැලේ බොන මැස්සන්ගෙන් හා, මදුරුවන්ගෙන් ස්පර්ශ කරණ ලද්දේ යුද පෙරමුණක ඇතෙකු මෙන් සිහි ඇතිව එහිම ඉවසන්නේය. 

# “බ්‍රහ්මයා යම්සේද එසේම තනියම හැසිරෙන්නේ සැපය. දෙවියන්ට සේ දෙදෙනෙකුවූ විට සිතේ කිපීම වන්නේය. තුන්දෙනෙකුගේ වාසය ගමේ විසීමක් මෙනි. තුන්දෙනෙකුට වඩා සිටින තැන කෝලාහලයකි.”
# ආයුෂ්මත් යසොජ තෙර මෙසේ මේ ගාථා කීයේය. 
# සාටිමත්තිය ථෙර ගාථා
# “තොපට පෙර ශ්‍රද්ධාවක් විය. ඒ ශ්‍රද්ධාව අද තොපට නැත. යමක් තොපගේද එය තොපගේමය. මගේ දුශ්චරිතයක් නැත. 
# “ශ්‍රද්ධාව අනිත්‍යය, සෙලවේ. මෙසේ මා විසින් දක්නාලදී. ඇලීමද නොඇලීමදවේ. එවැන්නක් සමග ඇලුණත් නොඇලුණත් ප්‍රව්‍රජිතයාට කිනම් පාඩුවක්ද, 
# “ගෙයක් ගෙයක් පාසා ප්‍රව්‍රජිතයාට බත් පිසියි. මගේ පාචල ශක්තිය ඇත. පිඬු පිණිස හැසිරෙන්නෙමි.”
# ආයුෂ්මත් සාටිමත්තිය තෙර මෙසේ මේ ගාථා කීයේය. 
# උපාලි ථෙර ගාථා
# “ශ්‍රද්ධාවෙන් ගෙන් නික්ම අළුත පැවිදිවූ නවක තෙමේ පිරිසිදු ජීවිකාව ඇති, කම්මැලි නොවූ යහපත් මිත්‍රයන් ආශ්‍රය කරන්නේය. 
# “ශ්‍රද්ධාවෙන් ගෙන් නික්ම අළුත පැවිදිවූ නවක තෙමේ සංඝයා කෙරෙහි වාසය කරණ ඤාණවන්ත මහණ තෙමේ විනයෙහි හික්මෙන්නේය. 
# “ශ්‍රද්ධාවෙන් ගෙන් නික්ම අළුත පැවිදිවූ නවක තෙමේ තෘෂ්ණාදිය පෙරටු නොකොට කැප ඇකැපයෙහි දක්ෂව හැසිරෙන්නේය.”
# ආයුෂ්මත් උපාලි තෙර මේ ගාථා මෙසේ කීයේය. 
# උත්තරපාල ථෙර ගාථා
# “අර්ථ කල්පනා කිරීමට සමර්ථවූ පණ්ඩිතවූ මා ලොකයෙහි පස්කම් රසය තෙමේ මුලාවට දැම්මේය. 
# “ඇනුණු තද හුලක් සමානව මාරයාගේ වසඟයට දුවන්නේ මර රජහුගේ තොණ්ඩුවෙන් බේරෙන්නට නොහැකිවිය. 

# “මාගේ සියලු කාමයෝ ප්‍රහීනය. සියලු භවයෝ ලිහා දමන ලද්දාහ. ජාතියෙන් ජාති ඉපදීම්යයි කියනලද පුනර්භවය දැන් නැත.”
# ආයුෂ්මත් උත්තරපාල තෙර මේ ගාථා මෙසේ කීයේය. 
# අභිභුත ථෙර ගාථා
# “යම්තාක් මෙහි රැස්වූ සියලු ඥාතීහු අසව්. තොපට ධර්මය දෙසන්නෙමි. නැවත නැවත ඉපදීම දුක්වෙයි. 
# “බුද්ධශාසනයෙහි ප්‍රථම වීර්යය කරව්. ඉන්වඩාත් වීර්ය කරව්. ශ්‍රෙෂ්ඨශීල ශ්‍රෙෂ්ඨ ශික්ෂාවෙහි යෙදෙව්. ඇතා හුණවනයක් මෙන් මාරසේනාව කම්පා කරව්. 
# “යමෙක් මේ ධර්ම විනයෙහි අප්‍රමාදව වසන්නේද උත්පත්තිය නමැති සසර හැර දුක්කෙළවර කරන්නේය.”
# ආයුෂ්මත් අභිභුත තෙර මේ ගාථා මෙසේ කීයේය. 
# ගොතම ථෙර ගාථා
# “හැසිරෙන්නේ නිරයට ගියෙමි. නැවත නැවත ප්‍රේත ලොකයට ගියෙමි. තිරිසන් යොනියෙහි දුකට ගියෙමි. නොයෙක් ප්‍රකාරයෙන් මෙතැන්හි බොහෝ කලක් මා විසින් වසනලදී. 
# “මා විසින් මනුෂ්‍යභවයද අමාරුවෙන් ලබන ලද්දේය. එක්වරක් දෙවරක් ස්වර්ග සම්පත්තියට පැමුණුනෙමි. රූපලොකයෙහිද අරූපලොකයෙහිද සංඥාව ඇති නැති තැන්හිද සංඥාව නැති තැනද උපන්නෙමි. 
# “හෙතු ප්‍රත්‍යයෙන් හටගත්දෙය, හරයක් නැති ලෙස දන්නාලදී. සංස්කාරයෝ ස්ථිරනොවූහ. බිඳෙන සුළුය තමන්ට අයත්වූ එය දැන සිහි ඇත්තේ නිවනම ලබන්නේය.”
# ආයුෂ්මත් ගොතම තෙර මෙසේ මේ ගාථා කීයේය. 
# හාරිත ථෙර ගාථා
# “ජරාවට පත්වීමට පෙර යමක් කළයුතුද එය ඔහු පසුව කරන්නට කැමතිවේද, හෙතෙම සැපයෙන් පහවේ. පසුවද තැවෙයි. 

# “යමක් කරන්නේද එය කියන්නේය. යමක් නොකරන්නේද එය නොකියන්නේය. පණ්ඩිතයෝ කියන්නාවූ දෙය නොකරන්නාහු විශෙෂයෙන් දැනගනිත්. 
# “යම්තැනක දුක නැතිවේද, බුදුන් විසින් වදාරණ ලද ශොක නැති, කෙලෙස් නැති, නිර්භය නිවන අතිශයින් සැපය.”
# ආයුෂ්මත් හාරිත තෙර මෙසේ මේ ගාථා කීයේය. 
# විමල ථෙර ගාථා
# “පාපමිත්‍රයන් දුරුකොට උතුම් පුද්ගලයන් ආශ්‍රය කරන්නේය. නොසෙල් වෙන සැපය පතමින් ඒ උතුම් පුද්ගලයාගේ අවවාදයෙහි සිටින්නේය. 
# “කුඩා දරකඩකට නැගී මහමුහුදෙහි යම්සේ ගිලේද එසේ කම්මැලියා වෙත පැමිණ යහපත්ව ජීවත්වන්නේද ගිලේ, එහෙයින් කම්මැලිවූ වීර්යය නැති ඔහු දුරුකරන්නේය. 
# “කාය විවේකයෙන් යුක්තවූ හා කෙලෙසුන්ගෙන් වෙන්වූ නිවන්වෙත යොමුකළ ජීවිත ඇති ධ්‍යානකරණ නිතර පටන්ගන්නා ලද වීර්යය ඇති පණ්ඩිතයන් හා වාසය කරන්නේය.
# ආයුෂ්මත් විමලතෙර මෙසේ මේ ගාථා කීයේය.
