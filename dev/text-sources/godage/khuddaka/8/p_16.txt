
තිංස නිපාතය 
# ඵුස්ස ථෙර ගාථා
###
# “අනාගත කාලයෙහි ක්‍රොධකරන්නාවූද බද්ධ වෛරයෙන් යුක්තවුනාවූද ගුණමකු බැව් ඇත්තාවූද තද බැව් ඇත්තාවුද කෛරාටික ගති ඇත්තාවූද ඊර්ෂ්‍යා කරන්නාවූද කළහකාරීවූද බොහෝ අය ඇතිවන්නාහ. 
# “ධර්මය නොදැනම මාන්නයෙන් යුක්තව අගෝචරයෙහි වෙසෙමින් චපලගති ඇතිව ගෞරව නැතිව වෙසෙත්. 
# “අනාගතයෙහි නොයෙක් දෝෂයෝ සත්වලෝකයෙහි ඇතිවන්නාහ. මනාකොට දේශනා කරනලද සද්ධර්මය මෝඩයෝ කෙලෙසත්. 
# “ගුණයෙන් හීන පුද්ගලයෝද සංඝයා මැද විනිශ්චය කරන අවස්ථාවෙහිදී නිර්භයව බලවන්තව රළුවචන ඇත්තෝව ඒ අසා ධර්මය අධර්මය කොට පෙන්වත්. 

# “ගුණ ඇත්තෝද සංඝයා මැද අර්ථානුකූලව පෙන්වනු ලබන කල්හි දුර්වලයෝව සිටිත්. ලජ්ජා ඇත්තෝව තුෂ්ණිම්භූත වෙත්. 
# “අනාගත කාලයෙහි රන්රිදීද, කෙත්වතුද, එළු බැටළු ආදී සතුන්ද, දැසිදසුන්ද, මෝඩ පුද්ගලයෝ පිළිගනිත්. 
# “අනුන් වෙහෙසනු කැමැත්තාවූ අඥානයෝ සිල් ගුණයක් නැතිව ඔසවනලද මාන්නයෙන් යුක්තව කලහ කරනු කැමැති මෘගයන් මෙන් හැසිරෙත්. 
# “උඩඟු ගතියෙන් යුක්තව අකැපවූ සිවුරු පොරවා කුහක ගතිවලින්ද, තද ගතියෙන්ද, කුහක වචනවලින්ද අන් හා සමාන ප්‍රකට ක්ලේශයන්ගෙන් යුක්තව හැසිරෙන්නාහුය. 
# “තෙලින් තෙමූ කෙසින් යුක්තව චපලගති ඇතිව අඳුන්වලින් අලංකාර කළ ඇස් ඇතිව ලා රතින් යුක්තව සිවුරු පොරවා රථවල නැගී හැසිරෙත්. 
# “ආර්යයන් විසින් පිළිකුල් නොකරනලද අර්හත් ධජය සුදට හුරු සිවුර නිසා පිළිකුල් කරත්. 
# “කුසීතවූද, හරණාලද වීර්ය ඇත්තාවූද, ඔවුන් ලාභයටම කැමති වෙත්. ඔවුන් වනයෙහි විසීමට අකැමැතිව ගම් සෙනසුන්වලම වෙසෙත්. 
# “යම් කලෙක මිච්ඡා ජීවිතයෙහිම ඇලී යම් යම් ලාභ ලබත්ද, ඒ ඒ පුද්ගලයින්ට අනුවම අසංවරව හැසිරෙත්. 
# “යම් යම් ලාභ නොලැබීමෙන් අලාභවලින් යුක්තවූ පූජ්‍ය ගුණ ඇති අය අනාගතයෙහි ඇති නොවෙත්. ධෛර්ය සම්පන්න ප්‍රිය දැකුම් ඇති භික්ෂූහුද, ඒ කාලයෙහි ලාභ කැමතිවම වෙසෙත්. 
# “කාලයට සුදුසු පඬු පෙවීමෙන් යුක්තවූ ස්වකීය රතු සිවුරට ගරහා තීර්ථකයින්ගේ සුදු සිව්ර දරාගනිත්. අනාගතයෙහි ගිහියෝද කසාවතට අගෞරව කරත්. භික්ෂූහුද ප්‍රත්‍යවෙක්ෂාවෙන් තොරව කසාවත් දරාගනිත්. 
#
# “හීයෙන් විදුනාලදුව දුකෙන් පෙළෙමින් ශරීර පීඩාවට පත් මහා ඡද්දන්ත හස්තිරාජයාද මහා ඝොරවූයේ නමුදු, 
# “අර්හත් ධජය දැක ඒ ඡද්දන්තහස්තියා විසින් අර්ථයෙන් යුක්ත මේ ගාථාවන් එකෙනෙහිම ප්‍රකාශ කරනලදී. 
# “යමෙක් ඉන්ද්‍රිය දමනයෙන් හා වාක් ශක්තියෙන් තොරව කසාවත් දරත්ද හෙතෙම කසාවතට නොහොබී. 
# “යමෙක් වනාහි කෙලෙස් කසට දුරුකොට සීලයෙන් යුක්තවූයේ ඉන්ද්‍රිය දමනයෙන් හා වාක් ශක්තියෙන් යුක්තවූයේවේද හෙතෙම කසාවතට සුදුසුය. 
# “සිල් බිඳුනාවූද, නුවණමදවූද, ප්‍රකට ඉඳුරන් ඇත්තාවූද, කැමති කැමති පරිදි හැසිරෙන්නාවූද, රූපාදී විෂයයන්හි විසුරුණාවූ සිත් ඇත්තාවූද, හිරිඔතප් නැත්තාවූද ඒ පුද්ගලයා කසාවතට නොහොබී. 
# “යමෙක් සීලයෙන් යුක්තද, රාගය දුරුකළේද, එකඟවූ සිත් ඇත්තේද පිරිසිදුවූ කල්පනා ඇත්තේද ඔහු කසාවතට සුදුසුය. 
# “යමෙකුන්ට සීලයක් නැත්තේද, උඩඟුවූද, මාන්නය ඇත්තාවූද, අඥානවූද, ඒ පුද්ගලයින්ට කසාවතින් කවර ප්‍රයෝජනයක්ද? 
# “දූෂිත සිත් ඇත්තාවුද, ආදර නැත්තාවූද භික්ෂූහුද, භික්ෂුණියෝද අනාගතයෙහි මෛත්‍රී සිත් ඇති යහපත් අයට නිග්‍රහ කරත්. 
# “මහතෙරුන් විසින් සිවුරු පෙරවීම් ආදී ලොකු කුඩා කටයුතුවල හික්මවන්නාහුද අඥානවූ ප්‍රකට ඉඳුරන් ඇති තමා කැමැතිදේම කරන්නාවූ මෝඩවූ ඒ පුද්ගලයෝ ඒ උතුම් අවවාද නොඅසත්. 
# “ආචාර්ය උපාධ්‍යායයන් විසින් හික්මවනු ලබන්නා වූ ඒ අඥාන පුද්ගලයෝ ඔවුනොවුන් කෙරෙහි ආදර නැතිව දුෂ්ට අසුන් මෙන් හැසිරෙත්. 

# “අනාගතයෙහි ඇතිවන භික්ෂු භික්ෂුණීන්ගේ ප්‍රතිපත්තිය එබඳුවන්නේය පසුකාලයක ඇතිවන වෙනස අමුතු ආකාරයකින් ඇතිවන්නේය. 
# “ඒ අනාගතයෙහි ඇතිවන මහාභයට පළමුව තොප විසින් සුවචව ඔවුනොවුන් කෙරෙහි ගෞරව ඇත්තෝව වාසය කරව්. 
# “මෛත්‍රී සිත් ඇත්තෝව, කාරුණිකයෝව සීල සංවරයෙන් යුක්තව වීර්ය ඇතිව නිවන් සොයන්නෝව සියලු කල්හි ස්ථිර වීර්යයෙන් යුක්තව වසව්. 
# “ප්‍රමාදය භය වශයෙන්ද අප්‍රමාදය ක්ෂෙම වශයෙන්ද දැන ආර්ය අෂ්ටාංගික මාර්ගය වඩව්. එසේ කල්හි නිර්වාණය ස්පර්ශ කරන්නාහුය.” 
#
# සාරිපුත්ත ථෙර ගාථා
# “සීල සම්පන්නවූද, ශාන්තවූද සිහියෙන් යුක්තවූද භවයෙන් නික්මීමේ කල්පනා ඇතිව ධ්‍යාන කරන්නාවූද අප්‍රමාද විහරණයෙන් යුක්තවූද, කර්ම ස්ථානභාවනාවෙහි ඇලුනාවූද, එකඟවූ සිත් ඇත්තාවූද විවේකය වඩන්නාවුද ප්‍රත්‍යයාදී සන්තොෂයෙන් යුක්තවූද, යම් පුද්ගලයෙක් වේ නම් ඔහු භික්ෂුයයි කියනු ලැබේ. 
# “සිනිඳුවූද, රළුවූද අහර බුදිනු ලබන්නේ බොහෝ කොට නුබුදීද ස්වල්පවූ ආහාරයෙන් නොපිරුණු බඩ ඇතිව සිහියෙන් යුක්තව වාසය කරන්නේය. 
# “සතර පස් පිඩකට ඉඩ තබා පැන්බොන්නේවේද, නිවනට මෙහෙයවූ සිත් ඇති භික්ෂුවගේ එය සැප විහරණය පිණිස වෙයි. 
# “සීතාදියෙන් වළකාසිටීමාදී ප්‍රයෝජන සලකා සැප සිවුරෙන් ශරීරය වසාද නිවන් පිණිස මෙහෙයූ සිත් ඇති භික්ෂුවගේ එය සැප විහරණය පිණිස වෙයි. 
# “එරමිණියා ගොතාහුන්නහුට වැස්සෙන් දෙදණ නොතෙමීසිටිය හැකිවේද, නිවන් පිණිස මෙහෙයවූ සිත් ඇති භික්ෂුවගේ එය සැප විහරණය පිණිස වෙයි. 

# “යමෙක් සුඛවේදනාව වෙනස්වන දුක් ඇති දෙයකැයි දකීද, දුක්වේදනා වනාහි හුල් වශයෙන් දකීද, මධ්‍යස්ථ වේදනාවෙහි මමය මාගේයයි යන හැසිරීමක් නැත්තේද, ලෝකයෙහි කිසිවක් නැති පුද්ගලයා ඔහු වන්නේය. 
# “යමෙක් පාපිච්ඡවේද, කුසීතවේද, වීර්ය අඩුකෙනෙක් වේද, අල්පශෘතයෙක්වේද, අනාදරයෙක්වේද, එබඳුවූ ලාමක පුද්ගලයා කිසිකලෙකත් මා කෙරෙහි නොවේවා. 
# “බහුශ්‍රැතවේද, ප්‍රඥාව ඇත්තාවුද, සීලයෙහි පිහිටියාවූද, සිත තැන්පත් කළාවූද පුද්ගලතෙම මාගේ මස්තකයෙහි සිටීවා. 
# “තෘෂ්ණාදී ප්‍රපංච ධර්මයන්ගෙන් යුක්තවූද ඒ ප්‍රපංච ධර්ම විෂයෙහිම ඇලුනාවූද, යම් පුද්ගලයෙක් මෘගයෙක් හා සමානවේද, ඒ පුද්ගලයා නිරුත්තරවූ යොගක්ෂෙම නම්වූ නිවණ නොලබයි. 
# “යම් පුද්ගලයෙක් තෘෂ්ණා ප්‍රමාද හැර පමා බවක් නැති නිවනෙහි ඇලුනේද ඒ පුද්ගලයා නිරුත්තරවූ යොගක්ෂෙම නම්වූ නිවන ලබයි. 
# “ගමෙහි හෝ අරණ්‍යයෙහි හෝ පහත් ස්ථානයෙහි ගොඩ තැන යන යම් ස්ථානයක රහතුන් වෙසෙත් නම් ඒ ස්ථානය රමණීය ස්ථානයක්වේ. 
# “යම් තැනක රමණීය වනයෝ වෙත්ද, ජනයා නොඇලෙන ඒ වනාන්තරවල වීතරාගී උතුමෝ ඇලෙත්. කාමය සොයන පුද්ගලයෝ නොඇලෙත්. 
# “යම් පුද්ගලයෙක් වරද දක්වන්නේ නම් ඔහු නිධාන් දක්වන්නෙක් වැනිය. නිග්‍රහ මුඛයෙන් කියන්නාට ප්‍රඥාව ඇති එබඳුවූ පණ්ඩිතයෝ භජනය කළයුතුය. එබන්දන් සේවනය කරන්නහුට ශ්‍රෙෂ්ඨ භාවය ඇතිවෙයි. ලාමක බවක් ඇති නොවෙයි. 
# “එම නිසා අවවාද කළයුතුයි. අනුශාසනා කළ යුතුයි. අසභ්‍ය භාවයෙන්ද වැළකිය යුතයි. එබඳු පුද්ගලයෝ සත්පුරුෂයන්ට ප්‍රියයි. අසත්පුරුෂයන්ට අප්‍රියයි. 

# “පසැස් ඇති බුදුරජාණන් වහන්සේ අනිකෙකුට යම් ධර්මයක් දේශනා කෙළේද ඒ ධර්මය දේශනා කරන කල්හි කන් නමා සිටියෙමි. 
# “මගේ ඒ ඇසීම් හිස්නොවූයේය. ආශ්‍රව රහිතව කෙලෙසුන්ගෙන් මිදුනෙමි. මගේ සිත පෙරවිසූ පිළිවෙල දැනීම පිළිබඳ නොවේ. දිවැස් ලැබීම පිණිසද නොවේ. 
“මාගේ ප්‍රාර්ථනාව වනාහි චුති උත්පත්ති දැනීමේ ඍද්ධිය පිළිබඳව හෝ දිව්‍යවූ කන් ලැබීම පිණිස හෝ නොවේ. 
# “හිස මුඩුකරන ලද්දේ සඟල සිවුර දැරුවේ රුක් මුල් ඇසුරුකොට ප්‍රඥාවෙන් උත්තම බවට පැමිණියේ උප තිස්සත්ථෙර නමින් ධ්‍යාන කරන්නෙහිය. 
# “සම්බුද්ධ ශ්‍රාවකවූ මම එකෙණෙහිම චතුර්ථධ්‍යාන සමාපත්තියට සමවැදී ආර්ය තුෂ්ණිම් භාවයෙන් යුක්තව සිටියෙමි. 
# “යම්සේ පර්වතය යහපත්ව පිහිටියේ නොසෙල්වේද එසේම මෝහය දුරුකළ භික්ෂුවද පර්වතයක් මෙන් නොසෙල්වෙයි. 
# “පිරිසිදුකම කැමැති කෙලෙස් රහිත පුද්ගලයාට ස්වල්පමාත්‍ර පාපයද මහත්කොට වැටහෙයි. 
# “ජීවිතයද කැමති නොවෙමි. මරණයද කැමති නොවෙමි. සිහියෙන් හා ප්‍රඥාවෙන් යුක්තව මේ ශරීරය බහා තබමි. 
# “මරණයටද කැමති නොවෙමි. ජීවිතයට කැමැති නොවෙමි. නිලයෙන් පහ කළ භටයෙක් මෙන් කාලය අපේක්ෂා කරමි. 
# “පෙරවරු පස්වරු යනාදී උභය කාලයන් අතර මරණය සිදුවේමය. නොමැරීමක් නම් නැත. එම නිසා ප්‍රතිපත්තිය පුරව. අපා දුකෙන් විනාශ නොවෙව. ක්ෂණ සම්පත්තිය නොයික්මවව. 

# “ඇතුළත හා පිටත ආරක්ෂාකරන ප්‍රත්‍යන්ත නගරයක් මෙන් ආත්ම භාවය ආරක්ෂා කරව. ක්ෂණ සම්පත්තිය පැහැර නොහරිව්. ඒ පැහැර හැරියේ නම් නිරයේ දුකට පත්වෙයි. 
# “ශාන්ත ගුණයෙන් යුක්තවූද එහිම ඇලුනාවූද ස්වල්ප බස් ඇත්තාවූද උඩඟු නොවූවාවූද පුද්ගලයා සුළඟින් ගසක කොළ හෙළන්නාක් මෙන් පාප ධර්මයන් බැහැර කරයි. 
# “ශාන්ත ගුණයෙන් යුක්තවූද එහිම ඇලුනාවූද ස්වල්ප බස් ඇත්තාවූද උඩඟු නොවූවාවූද පුද්ගලයා සුළඟින් ගසක කොළ හෙළන්නාක් මෙන් පාප ධර්මයන් අළුත පහකළේය. 
# “ශාන්ත ගුණයෙන් යුක්තවූයේ ක්ලේශ දුක් නැත්තේ පිරිසිදුවූයේ නොකැළඹුනේ යහපත් සිල් ඇත්තේ ප්‍රඥාවෙන් යුක්තවූවෙක්වේද හෙතෙම දුක් නැතිකරයි. 
# “අගාරියවූද අනගාරියවූද එකද පෘථග්ජනයෙකු කෙරෙහිවත් විශ්වාසය නොකටයුතුය. මක්නිසාදයත් යහපත් තැනැත්තෙක්ව අයහපත් තැනැත්තෙක්ද වෙයි. අයහපත් වූවෙක්ව යහපත්වූවෙක්ද වෙයි. 
# “කාමාශාවද ක්‍රොධයද ථිනමිද්ධයද නොසන්සුන් බවද සැකයද යන මේ පස භික්ෂුවගේ සිත කෙලෙසයි. 
# “යමෙක්හට සත්කාරයෙහිද අසත්කාරයෙහිද සිත වෙනස් නොවේ. තම අප්‍රමාදී විහරණයෙන් යුක්තවූ 
# “ඒ පුද්ගලයා නිතර ධ්‍යාන කරන්නේවේද විදර්ශනා වැඩීමෙන් උපාදානයන් ක්ෂය කිරීමේ ඇලුනාවූ ඔහු සත්පුරුෂයෙක් නොවේ. 
# “උත්තම මිදීමෙන් යුක්තවූ බුදුන් හා මහා සමුද්‍රයද පෘථිවියද පර්වතයෝද ආකාශයද සමකළ හැකිය. 
# “ශාස්තෲන් විසින් දෙසනලද ධර්මයට අනුව ඒ ධර්මය පවත්වන්නාවූ මහා ඤාණයක් ඇති එකඟවූ සිත් ඇති උන්වහන්සේ ඉෂ්ටානිෂ්ටාදි අරමුණු විෂයෙහි නොඇලෙයි. දූෂ්‍යද නොවෙයි. 

# “ශ්‍රාවක ඥානයෙහි පරතෙරට පත්වුනාවූද මහා බුද්ධියක් ඇත්තාවූද, මහා ප්‍රඥාවක් ඇත්තාවූද උන්වහන්සේ ගොළු නොවූයේ ගොළුවෙක් මෙන් අඟවන තැනැත්තෙක්ව වෙසෙයි. 
# “මා විසින් ශාස්තෲන් ඇසුරු කරන ලදී. බුදුන්ගේ අනුශාසනාවට අනුව කටයුතු කෙළෙමි. ගරු භාරය බහා තැබීමි. දැන් මට පුනර්භවයක් නැත්තේය. 
# “අප්‍රමාදයෙන් සම්පාදනය කරව්. මේ මාගේ අනුශාසනාව වන්නේය. මම සියල්ලෙන් මිදුනෙකුව පරිනිර්වාණය කරන්නෙමි.”

# ආනන්ද ථෙර ගාථා
# “පිසුණු බස් කීමෙන්ද කිපෙන ගතියෙන්ද මසුරු කමින්ද විනාශය කැමතිවීමෙන්ද යුක්තවූ පුද්ගලයන් සමග සංසර්ග නොකටයුතුය. පණ්ඩිතයා ලාමකයන් සමග නොවෙයි. 
# “අදහන්නාවූද සීලයෙන් යුක්තවූද ප්‍රඥාව ඇත්තාවුද බහුශ්‍රැතවූද පුද්ගලයින් හා සහාය භාවය ඇතිකර ගතයුතුය. යහපත් ගුණ ඇත්තේ සත්පුරුෂයන් සමග එක්වෙත්. 
#වස්ත්‍රාභරණාදියෙන් විසිතුරු කරනලද නවමුඛයකින් අසූචි වැගිරෙන තුන්සීයක් ඇට කෝටු ඇති නිතර ගිලන් ස්වභාවයෙන් යුක්තවූ බාල ජනයන් විසින් වරදවා කල්පනා කරන ස්ථිර භාවයක් නැති බිඳෙන ස්වභාවයෙන් යුක්තවූ මේ ශරීරය බලව. 
# “මාණික්‍යාදි ආභරණ විශෙෂයන්ගෙන් විසිතුරු කරන ලද්දාවූද ඇට සහ සමෙනුත් යුක්තවූද වස්ත්‍රයන්ගෙන් ශොභා සම්පන්න මේ ශරීරය බලව. 
# “ලාකඩ වතුර රැඳවූ විලේපනවලින් ඔපකරන ලද මුඛයක් ඇති මේ ශරීරය අඥාන පුද්ගලයාගේ මුලාවට හේතු වෙයි. නිවන් සොයන්නහුට මුලාවට කාරණයක් නොවේ. 
# “අෂ්ටපදාකාරයෙන් කරනලද කෙස් ඇත්තාවූද, අඳුන් තැවරූ ඇස් ඇත්තාවූද මේ ශරීරය අඥානයාගේ මුලාවට හේතුයි. නිවන් සොයන්නහුගේ මුලාවට හේතුවක් නොවේ. 
# “අලංකාර කරනලද මේ කුණු කය මාලාකර්ම දන්තාදියෙන් විසිතුරුය. ඒ බාලයාගේ මුලාවට හේතුවක් වෙයි. 
# “බහුශ්‍රැතයෙක්වද විසිතුරු කථිකයෙක්වද බුදුන්ට උපස්ථාන කරන්නෙක්වද සියලු යෝගයන්ගෙන් වෙන්ව බහා තබන ලද බර ඇත්තේ සයනය කරන්නෙමි. 
# “නැති කරන ලද ආශ්‍රව ඇත්තේ රාගාදී ඇලීම් ඉක්මවන ලද්දේ ක්ලේශ පරිඩාභයන් දුරුකරන ලද්දේ සිහිල්වූයේ ජාති මරණ ඉක්මවන ලද්දේ අන්තිම ශරීරය දරමි. 
# “සූර්යවංශයෙහි උපන් බුදුන්ගේ යම්බඳුවූ ධර්මයක පිහිටියේද ඒ නිර්වාණ ධර්මයෙහි මම පිහිටා සිටිමි. 
# “දෙ අසූදාහක් අධිකකොට ඇති සූ අසූ දහසක් ධර්මයෝ බුදුන්ගෙන් මම දැනගතිමි. ඒ ධර්මයෝ මා ලග පවතින්නාහ. 
# “අල්පශ්‍රැත පුද්ගලයා ගවයෙකු මෙන් ජීවත්වෙයි. ඔහුගේ ශරීරය මස් වලින් වැඩෙයි. ප්‍රඥාවෙන් නොවැඩෙයි. 

# “යම් බහුශ්‍රැත පුද්ගලයෙක් අල්පශ්‍රැත පුද්ගලයෙකු අවමන් කොට සිතයිද හෙතෙම පහනක් දරන්නාවූ අන්ධයෙක් මෙන් වැටහෙයි. 
# “බහුශ්‍රැත පුද්ගලයෝ සෙවිය යුතුය. ශ්‍රැතවත්භාවය නොනැසිය යුතුය. ඒ බ්‍රහ්මචර්යාවට මුලවන්නේය. එහෙයින් ධර්මධරයෙක්ම විය යුතුය. 
# “පෙර අපර අර්ථ දන්නාවූ නිරුක්ති පදයන්හි දක්ෂවූ පුද්ගලයා ධර්මය මනාකොට ගණියි. අර්ථද විමංසනය කරයි. 
# “විදර්ශනාවෙහි හටගත් ඡන්දය ඇත්තේ උත්සාහයෙන් ත්‍රි ලක්ෂණයට ආරොපණයකොට තීරණය කරයි. එසේ බලන්නාවූ හෙතෙම සිත එකඟ කෙලෙස් දුරුකරයි. 
# “බහුශ්‍රැතවූද සප්‍රාඥවූද බුද්ධශ්‍රාවකයන් සෙවිය යුතුය. එබඳුවූ උතුමන් සෙවීමෙන් මනාකොට ධර්මය දත හැකිය. 
# “බහුශ්‍රැතවූද ධර්ම ගබඩාව ධර්ම කොෂය ආරක්ෂා කරන්නාවූද පුද්ගලයා ලොවට ඇසක් බඳුය. බහුශ්‍රැතයන් විසින් පුදනලද්දෙක්ද වේ. 
# “ධර්මය අරමුණකොට ඇත්තාවූද ධර්මයෙහි ඇලුනාවූද ධර්මයට අනුව සිතන්නාවූද ධර්මය සිහිකරන්නාවූද භික්ෂුව සද්ධර්මයෙන් නොපිරිහෙයි. 
# “කාය මමත්වයෙහි ඇලී කිසිවක් නොකරන තැනැත්තා පිරිහෙනුයේ උට්ඨාන වීර්යයෙන් තොරව ශරීර සැපයෙහිම ඇලී වෙසේ නම් ඔහුට ශ්‍රමණ සැපය කොයින්ද? 
# “සියලු දිශාවන් නොපෙනෙන්නාක් මෙන් ධර්මය නොපෙනේද කල්‍යාණ මිත්‍රයා ගිය කල්හි අන්ධකාරයක් මෙන් වැටහෙයි. 
# “කල්‍යාණ මිත්‍රයන් නැත්තාවූද බුදුන් පිරිනිවියාවූද කල්හි කායගතා සති භාවනාව හා සමවූ කල්‍යාණ මිත්‍රයෙක් නැත්තේය. 
# “පුරාණ මිතුරන් නැත්තේ මගේ සිත එකඟ නොවනුයේ කැදැල්ලට පැමිණි පක්ෂියෙකු මෙන් හුදකලාව සිතිවිලි සිතයි. 

# “මගේ දැකීම නැතිකල්හි නොයෙක් දිශාවලින් පැමිණෙන්නෝ නොවලක්වව. අසනු කැමැත්තන්ට මගේ ධර්මය පෙන්නුම් කරත්වා. 
# “දකින්ට නැතිකල්හි නොයෙක් රාජ්‍යවලින් පැමිණි බොහෝ දෙනාට අවකාශ ඇති කරවා ශාස්තෲන් වහන්සේ ඔවුන් නොවළක්වයි. 
# “පස්විසි හවුරුද්දක් හික්මෙන භූමියෙහි සිටි මට කාමසංඥාවෝ ඇති නොවීය. ධර්මයාගේ යහපත් බව බලව. 
# “පස්විසි හවුරුද්දක් හික්මෙන භූමියෙහි සිටි මට ද්වෙෂ සහගත හැඟීම් ඇති නොවීය. ධර්මයාගේ යහපත් බව බලව. 
# “පස්විසි හවුරුද්දක් අනුව පවතින ඡායාවක් මෙන් මෛත්‍රී කායකර්මයෙන් යුක්තව භාග්‍යවතුන් වහන්සේට උපස්ථාන කෙළෙමි. 
# “පස්විසි හවුරුද්දක් අනුව පවතින ඡායාව මෙන් මෛත්‍රී වාක් කර්මයෙන් යුක්තව භාග්‍යවතුන් වහන්සේට උපස්ථාන කෙළෙමි. 
# “පස්විසි හවුරුද්දක් අනුව යන ඡායාවමෙන් මෛත්‍රී සහගත මනෝ කර්මයෙන් යුක්තව භාග්‍යවතුන් වහන්සේට උපස්ථාන කෙළෙමි. 
# “බුදුන් වඩින්නාවූ මග මමද පසුපසින් ගියෙමි. උන්වහන්සේ දහම් දෙසන කල්හි මට ඤාණය පහළවිය. 
# “මම කටයුතු ඇත්තෙක්වීමි. නොලත් රහත් බැව් ඇත්තේද වීමි. ශාස්තෲන් වහන්සේගේ පරිනිර්වාණයද එළඹිය ඒ අවස්ථාවෙහිද මට බුදුහු අනුකම්පා කළසේක. 
# “බුදුන් පිරිනිවිකල්හි බියජනකවූ රෝමෝද්ගමනය ඇතිකරන්නාවූ පෘථිවි කම්පාදිය සිදුවිය. 
# “මහර්ෂීවූ බුදුන්වහන්සේගේ ධර්මභාණ්ඩාගාරය ආරක්ෂා කළ බහුශ්‍රැත ධර්මධර සියලු ලෝකයාගේ ඇසක් බඳුවූ ආනන්ද ස්ථවිරයන් වහන්සේ පිරිනිවන් පෑසේක. 

# “මහර්ෂීවූ බුදුන්ගේ ධර්මය ආරක්ෂා කළ බහුශ්‍රැත ධර්මධර ඇසක් බඳුවූ ආනන්ද ස්ථවිරයන්ගේ පිරිනිවන්පෑම ලෝකයෙහි අන්ධකාරයට හේතුවිය. 
# “ඥාන ගතියෙන් යුක්තවූද ස්මෘති ප්‍රඥාවෙන් යුක්ත වූද ධෛර්යවත්වූද යම් ඍෂිවරයෙක් සද්ධර්මය දැරුවේනම් ඒ ආනන්ද ස්ථවිරයන් වහන්සේ රත්නාකාරයක් බඳුය. 
# “මා විසින් බුදුන් ඇසුරුකරන ලදී. බුදුන්ගේ අනුශාසනාවද කෙළෙමි. ගරු භාරය බහා තැබීමි. පුනර්භවයක් මට නැත්තේය.”
