සත්තක නිපාතය. 
# සුන්දර සමුද්ද ථෙර ගාථා
# “මනාකොට වැසීමෙන් අලංකාර කරනලද සිනිඳු මල්දම්වලින්ද සරසන ලදුව ලාකඩ රසයෙන් ලක්ෂණ කළ පා දෙකින් යුක්තවූ ස්වර්ණමය පාවහන් ඇති වෙශ්‍යා තොමෝ 
# “පාවහන් ගලවා ඉදිරිපිට ඇඳිලි බැඳගෙණ ( ඕ තොමෝ) මට මධුරවූ වචනයෙන් මෙසේ කීවාය. 

# “තෝ බාල වියෙහිදීම පැවිදිවූවෙහිය. මාගේ වචනය කරන්නෙහි නම් මානුෂික කාමයන් පරිභොග කරව. ඒ පිණිස මගේ සිත ඔබට දෙමි. 
# “ඔබ විසින් මෙය ඇත්තක්කොට පිළිගනුව. ඒ අදහන්නහු නම් ගිනි ඉදිරිපිට තබා දිවුරා කියනු. 
# “අප දෙදෙනාම සැරයටිය පිහිටකොට පවත්නා කල්හි පැවිදි වන්නමු නම් දෙපැත්තෙන්ම ජයගත්තේවේ. වෙශ්‍යාංගනාව විසින් ඇදිලි බැඳ කළ මේ ආයාචනය දැකීමෙන්, 
# “මාරයා විසින් ඔසවනලද පාසයක් (උගුලක්) බඳුවූ අලංකෘත වස්ත්‍රාභරණ සහිත වෙශ්‍යාව දැකීමෙන් මට සිහිනුවණ උපණ. 
# “දොෂයන් ප්‍රකටව පෙණින, කලකිරීමද ලබනලදී. එයින්ම සිත පස්වැදෑරුම් විමුක්තියෙන් මිදුණි. ධර්මයාගේ යහපත් බව ලබව. ත්‍රිවිද්‍යාව ලදිමි. බුදුන්ගේ අනුශාසනාවද කරන ලදී. 
#
# ලකුණ්ඨක භද්දිය ථෙර ගාථා
# “ශ්‍රෙෂ්ඨවූ අම්බට්ඨ කාරාමයෙහිවූ වන ලැහැබෙහි භද්දිය ස්ථවිරයන් වහන්සේ අවිද්‍යා සහිත තෘෂ්ණාව නසා එහිම සීලාදී ගුණයන්ගෙන් යහපත් බවට පැමිණි කෘත්‍යය ඇත්තේ මේ ආත්මයෙහිම ධ්‍යාන සමාපත්ති සැපයෙන් විසුවෙමි. 
# “මිහිඟුබෙර, වීණා, පනා බෙර යන නෘත්‍ය භාණ්ඩ විශෙෂයන්ගෙන් ඇතැම් කාමභොගී සත්වයෝ සතුටුවෙති. මමද ගස්මුල්වල බුදුන්ගේ අනුශාසනාව කරමින් සතුටුවෙමි. 
# “බුදුරජාණන් වහන්සේ මට වර දෙන ලද්දේද, ඒ වරය මම ලබාගතිමි. සියලු ලෝකයාට කායගතාසති කර්මස්ථානය වඩන පිණිසයි. 
# “ලාමකවූ ජනයා විසින් මගේ නිහීණ රූපය පිළිකුල් කොටද, පහත්කොටද, සිතුවාහුය, සමහර සත්වයෝ ගරු කිරීම් වශයෙන් මට බුහුමන් කළාහුය. ඒ දෙකොටසම ඡන්ද රාගයෙන් යුක්තවූවාහුය. මම ඡන්දරාගය නැත්තෙකු බව ඔව්හු නොදනිති. 
#
# “ආධ්‍යාත්මික තත්ත්වය නොදැක පිටස්තර තත්ත්වය දකීනම් හෙතෙම පිටස්තර බලය දැක්කාවූ ඔහු අනුන්ගේ කියමන් අනුවයේ. 
# “යමෙක් ආධ්‍යාත්මික තත්ත්වය දැන, බාහිර තත්ත්වයන් බලානම් යහපත් හැසිරීම් සහිත ඒ පුද්ගලයා අනුන්ගේ කියමනට නොනැමෙයි.” 
#
# භද්ද ථෙර ගාථා
# “මම බොහෝවූ ව්‍රත චරියාවෙන් හා ආයාචනාවලින්ද, මවුපියන් විසින් දුකසේ ලැබගත් එකම ප්‍රිය පුත්‍රයෙක්වීමි. 
# “යහපත කැමැති හිතෛෂිවූ ඒ මවුපිය දෙදෙනා මට අනුකම්පා උපදවා බුදුන්ට මා භාරකළහ. 
# “සැපසේ වැඩුවාවූද සුඛුමාලවූද දුකසේ ලැබගත් අපගේ පුත්‍රයා නාථ වූ බුදුරජාණන් වහන්ස, ඔබ වහන්සේට පවරා දෙමි. 
# “ශාස්තෲන් වහන්සේද මා පිළිගෙණ ආනන්ද ස්ථවිරයන් වහන්සේට මෙසේ වදාළහ. ‘මොහු වහා පැවිදි කරව, මෙතෙම ප්‍රධාන පුරුෂයෙක්ය.’ 
# “මා පැවිදි කරවා ශාස්තෲන් වහන්සේ විහාරයට පිවිසිසේක. හිර නොබැසි කල්හිම මගේ සිත කෙලෙසුන්ගෙන් මිදුනෙමි. රහත් බවද ලදිමි. 
# “ඉන්පසුව ශාස්තෲන් වහන්සේ ඵල සමවතින් නැගිට ‘භද්ද එව’ යන වචනයෙන්ම මගේ උපසම්පදාව සිදුවිය. 
# “උත්පත්තියෙන් සත්වෙනි හවුරුද්දෙහි මම උපසම්පදාව ලදිමි. ත්‍රිවිද්‍යාවද ලදිමි. ධර්මයක යහපත් බව ආශ්චර්යයි.” 
#
# සොපාක ථෙර ගාථා
# “ගඳකිළි සෙවනේදී සක්මන් කරන, මිනිසුන්ට උතුම්වූ බුදුරජාණන් වහන්සේ දැක එතැනට එළඹ උන්වහන්සේට වැන්දෙමි. 
# “සිවුර ඒකාංශකොට දෙඅත් එකතුකොට වීතරාගීවූ සියලුම සත්වයින්ට උතුම් බුදුන්ට අනුව මමද සක්මන් කෙළෙමි. 

# “ඉක්බිති සියල්ල දත් බුදුරජාණන් වහන්සේ මගෙන් ප්‍රශ්න ඇසීය. මමද බියක් හා තැතිගැණීමක් නැතිව ශාස්තෲන් වහන්සේට උත්තර දුනිමි. 
# “ප්‍රශ්න විසඳූ කල්හි තථාගතයන් වහන්සේ අනුමෝදන් විය. ඉක්බිති භික්ෂු සංඝයා බලා මේ කාරණය වදාළහ. 
# “අඟු මගධ දෙරට වැස්සන්ට ලොකු ලාභයෙකි. කුමක් හෙයින්ද ඔවුන්ගේ චීවර පිණ්ඩපාත සයනාසනාදී ප්‍රත්‍යය සොපාක ස්ථවිරයන් වහන්සේ පරිභොග කරන හෙයිනි. 
# “පෙර ගමන් කිරීම, පවන්සැලීමාදී ක්‍රියාවන්ගෙන්ද ඔවුන්ටවූයේ ලාභයක්යයි කියමි. අද පටන් සොපාක ස්ථවිරයන් වහන්සේ දැකීමට එළඹෙන්නෙහිය. සොපාකය, තොපට මේ ප්‍රශ්න ව්‍යාකරණය උපසම්පදාව වූවෙහිය. 
# “මම සත්වෙනි හවුරුද්දේදී උපසම්පදාව ලැබ අන්තිම ශරීරයද දරමි. ධර්මයාගේ යහපත් බව ආශ්චර්යයි.” 
#
#සරභංග ථෙර ගාථා
# “පෙර මම අතින් සිඳින තණ කුටියක් කොට එහි හොත්තෙමි. එයින් මට සරභංගයයි නාම සම්මුතිය විය. 
# “මා අද පටන් තෘණකුටි බැඳීම නොවටී. කුමක් හෙයින්දයත් කීර්තිමත් බුදුරජාණන් වහන්සේ විසින් ශික්ෂාපද පනවනලද හෙයිනි. 
# “සියල්ල ශෙෂනොකොට රෝගයන්ට හේතුවූ පඤ්චස්කන්ධය පෙර නොදැක්කෙමි. ඒ මම බුදුන්ගේ අවවාද අනුව කටයුතු කිරීම නිසා පඤ්චස්කන්ධ නම් වූ රෝගය නිසා විදර්ශනා සහිත මාර්ග ප්‍රඥාවෙන් දැක්කෙමි. 
# “යම් මාර්ගයකින් විපස්සී බුදුරජාණන් වහන්සේ ගියෙහිද, යම් මාර්ගයකින් සිඛී, කකුසඳ, කොණාගම කාශ්‍යප යන බුදුවරු ගියෝද, ඒ ආර්යමාර්ගයෙන්ම ගෞතම බුදුරජාණන් වහන්සේත් ගියසේක. 
# “පහකළ තෘෂ්ණාව ඇති ප්‍රතිසන්ධිය නැතිකළ සත් බුදුවරයෝ නිවන් ලැබුවාහුය. ඒ සත්බුදුවරයන් විසින් ශාසන ධර්මය ප්‍රකාශ කරන ලදහ. ධර්ම ස්වභාවයෙන්ම තාදීගුණ ඇත්තෝවූහ. 

# “දුක්ඛසත්‍යය, සමුදය, නිරොධ, මාර්ග යන දුක් නසන චතුරාර්ය සත්‍යධර්මය සත්වයින්ට අනුකම්පාවෙන් දෙශනා කරන ලදහ. 
# “නිවන ලැබූ කල්හි අනන්ත සංසාර දුක් නවතී. බිඳෙන මේ කය හා ජීවිත පරම්පරාව නවතී. නැවත ඉපදීමක් නැත. සියලු භවයන්ගෙන් මිදුනෙමි.” 
#
#සත්තක නිපාතය නිමි.
