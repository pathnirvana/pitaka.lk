
කිලෙස සංයුත්තය. 

#චක්ඛු සූත්‍රය
#[1] මා විසින් මෙසේ අසන ලදී. එක් කලෙක භාග්‍යවතුන් වහන්සේ සැවැත්නුවර සමීපයෙහිවූ අනේපිඬු සිටාණන් විසින් කරවන ලද ජේතවනාරාමයෙහි වාසය කරන සේක. එහිදී භාග්‍යවතුන් වහන්සේ ‘මහණෙනි’යි කියා භික්ෂූන් ඇමතූසේක. ‘ස්වාමීනි’යි කියා ඒ භික්ෂූහු භාග්‍යවතුන් වහන්සේට උත්තර දුන්හ. (එවිට) භාග්‍යවතුන් වහන්සේ මෙය වදාළ සේක. 
[2] “මහණෙනි, ඇසෙහි යම් ඡන්දරාගයක් වේද, එය චිත්තයාගේ කිලුටුවීමයි. කණෙහි යම් ඡන්දරාගයක් වේද, එය කණෙහි කිලුටුවීමයි. නාසයෙහි යම් ඡන්දරාගයක් වේද, එය නාසයෙහි කිලුටුවීමයි. දිවෙහි යම් ඡන්දරාගයක් වේද, එය දිවෙහි කිලුටුවීමයි. කයෙහි යම් ඡන්දරාගයක් වෙයිද, එය කයෙහි කිලුටුවීමයි සිතේ යම් ඡන්දරාගයක් වෙයිද, එය චිත්තයාගේ කිලුටුවීමයි. 
[3] “මහණෙනි, යම් තැනක පටන් මේ සය ස්ථානයන්හි භික්ෂුවගේ සිතේ කිලුටුබව ප්‍රහීනවූයේ වෙයිද? ඔහුගේ සිත ලොකොත්තර ධර්මයට නැඹුරුවූයේ වෙයි. නෛෂ්ක්‍රම්‍ය පුරුදු කරන ලද සිත විශෙෂ ඥානයෙන් ප්‍රත්‍යක්ෂ කළ යුතු ධර්ම විෂයෙහි කර්මණ්‍ය වෙයි. 

#රූප සූත්‍රය
#[1] (මේ සූත්‍රයේ 1, 3 වෙනි ඡෙද කිලෙස සංයුත්තයේ 1 චක්ඛු සූත්‍රයේ මෙනි.) 
[2] “මහණෙනි, රූපයෙහි යම් ඡන්දරාගයක් වේද, එය සිතේ කිලුටුවීමයි ශබ්දයෙහි යම් ඡන්දරාගයක් වේද, එය කණෙහි කිලුටුවීමයි. ගන්ධයෙහි යම් ඡන්දරාගයක් වේද, එය නාසයෙහි කිලුටුවීමයි. රසයෙහි යම් ඡන්දරාගයක් වේද, එය දිවෙහි කිලුටුවීමයි. ස්පර්ශයෙහි යම් ඡන්දරාගයක් වේද, එය කයෙහි කිලුටුවීමයි. ධර්මයෙහි යම් ඡන්දරාගයක් වේද, එය සිතේ කිලුටුවීමයි. 

#චක්ඛු විඤ්ඤාණය සූත්‍රය
#[1] (මේ සූත්‍රයේ 1, 3 වෙනි ඡෙද කිලෙස සංයුත්තයේ 1 චක්ඛු සූත්‍රයේ මෙනි.) 
[2] “මහණෙනි, චක්ඛු විඤ්ඤාණයෙහි යම් ඡන්දරාගයක් වේද, එය සිතේ කිලුටුවීමයි. සොත විඤ්ඤාණයෙහි යම් ඡන්ද රාගයක් වේද, එය කණෙහි කිලුටුවීමයි. ඝාණ විඤ්ඤාණයෙහි යම් ඡන්දරාගයක් වේද, එය නාසයේ කිලුටුවීමයි. ජිහ්වා විඤ්ඤාණයෙහි යම් ඡන්දරාගයක් වේද, එය දිවෙහි කිලුටුවීමයි. කාය විඤ්ඤාණයෙහි යම් ඡන්දරාගයක් වේද, එය කයෙහි කිලුටුවීමයි. මනෝ විඤ්ඤාණයෙහි යම් ඡන්දරාගයක් වේද, එය සිතේ කිලුටුවීමයි. 

#ඵස්ස සූත්‍රය
#[1] (මේ සූත්‍රයේ 1, 3 වෙනි ඡෙද කිලෙස සංයුත්තයේ 1 චක්ඛු සූත්‍රයේ මෙනි.) 
[2] “මහණෙනි, චක්ඛුසම්ඵස්සයෙහි යම් ඡන්දරාගයක්වේද, එය සිතේ කිලුටුවීමයි. සොත සම්ඵස්සයෙහි යම් ඡන්දරාගයක් වේද, එය කණෙහි කිලුටුවීමයි. ඝාණ සම්ඵස්සයෙහි යම් ඡන්ද රාගයක් වේද, එය නාසයේ කිලුටුවීමයි ජිව්හා සම්ඵස්සයෙහි යම් ඡන්දරාගයක්වේද, එය දිවෙහි කිලුටුවීමයි. කාය සම්ඵස්සයෙහි යම් ඡන්දරාගයක් වේද, එය කයෙහි කිලුටුවීමයි. මනෝ සම්ඵස්සයෙහි යම් ඡන්දරාගයක් වේද, එය සිතේ කිලුටුවීමයි. 

#වේදනා සූත්‍රය
#[1] (මේ සූත්‍රයේ 1, 3 වෙනි ඡෙද කිලෙස සංයුත්තයේ 1 චක්ඛු සූත්‍රයේ මෙනි) 
[2] “මහණෙනි, චක්ඛු ස්පර්ශයෙන් උපන් වේදනාවෙහි යම් ඡන්දරාගයක් වේද, එය සිතේ කිලුටුවීමයි. සොත ස්පර්ශයෙන් උපන් වේදනාවෙහි යම් ඡන්දරාගයක් වේද, එය කණෙහි කිලුටුවීමයි. ඝාණ ස්පර්ශයෙන් උපන් වේදනාවෙහි යම් ඡන්දරාගයක් වේද, එය නාසයේ කිලුටුවීමයි. ජිව්හා ස්පර්ශයෙන් උපන් වේදනාවෙහි යම් ඡන්දරාගයක් වේද එය දිවෙහි කිලුටුවීමයි. කාය ස්පර්ශයෙන් උපන් වේදනාවෙහි යම් ඡන්දරාගයක් වේද, එය කයෙහි කිලුටුවීමයි. මනො ස්පර්ශයෙන් උපන් වේදනාවෙහි යම් ඡන්දරාගයක් වේද, එය සිත පිළිබඳ කිලුටුවීමයි. 

#සඤ්ඤා සූත්‍රය
#[1] (මේ සූත්‍රයේ 1, 3 වෙනි ඡෙද කිලෙස සංයුත්තයේ 1 චක්ඛු සූත්‍රයේ මෙනි.) 
[2] “මහණෙනි, රූප සඤ්ඤාවෙහි යම් ඡන්දරාගයක් වේද එය සිතේ කිලුටුවීමයි. සද්ද සඤ්ඤාවෙහි යම් ඡන්දරාගයක් වේද, එය කණෙහි කිලුටුවීමයි. ගන්ධ සඤ්ඤාවෙහි යම් ඡන්දරාගයක් වේද, එය නාසයේ කිලුටුවීමයි. රස සඤ්ඤාවෙහි යම් ඡන්දරාගයක් වේද, එය දිවෙහි කිලුටුවීමයි. ඵොට්ඨබ්බ සඤ්ඤාවෙහි යම් ඡන්දරාගයක් වේද, එය කයෙහි කිලුටුවීමයි. ධම්ම සඤ්ඤාවෙහි යම් ඡන්දරාගයක් වේද, එය සිතේ කිලුටුවීමයි. 

#චේතනා සූත්‍රය
#[1] (මේ සූත්‍රයේ 1, 3 වෙනි ඡෙද කිලෙස සංයුත්තයේ 1 චක්ඛු සූත්‍රයේ මෙනි) 
[2] “මහණෙනි, රූප සංචේතනාවෙහි යම් ඡන්දරාගයක් වේද, එය සිතේ කිලුටුවීමයි. සද්ද සංචේතනාවෙහි යම් ඡන්දරාගයක් වේද, එය කණෙහි කිලුටු වීමයි. ගන්ධ සංචේතනාවෙහි යම් ඡන්දරාගයක් වේද, එය නාසයෙහි කිලුටුවීමයි. රස සංචේතනාවෙහි යම් ඡන්දරාගයක් වේද, එය දිවෙහි කිලුටු වීමයි. ඵොට්ඨබ්බ සංචේතනාවෙහි යම් ඡන්දරාගයක් වේද, එය කයෙහි කිලුටු වීමයි. ධර්ම සංචේතනාවෙහි යම් ඡන්දරාගයක් වේද, එය සිතේ කිලුටුවීමයි. 

#තණ්හා සූත්‍රය
#[1] (මේ සූත්‍රයේ 1, 3 වෙනි ඡෙද කිලෙස සංයුත්තයේ 1 චක්ඛු සූත්‍රයේ මෙනි.) 
[2] “මහණෙනි, රූපතණ්හාවෙහි යම් ඡන්දරාගයක් වේද එය සිතේ කිලුටුවීමයි. සද්ද තණ්හාවෙහි යම් ඡන්දරාගයක්වේද, එය කණෙහි කිලුටු වීමයි. ගන්ධ තණ්හාවෙහි යම් ඡන්ද රාගයක් වේද, එය නාසයෙහි කිලුටුවීමයි. රස තණ්හාවෙහි යම් ඡන්දරාගයක් වේද, එය දිවෙහි කිලුටුවීමයි. ඵොට්ඨබ්බ තණ්හාවෙහි යම් ඡන්දරාගයක් වේද, එය කයෙහි කිලුටු වීමයි ධම්ම තණ්හාවෙහි යම් ඡන්දරාගයක් වේද, එය සිතේ කිලුටු වීමයි. 

#ධාතු සූත්‍රය
#[1] (මේ සූත්‍රයේ 1, 3 වෙනි ඡෙද කිලෙස සංයුත්තයේ 1 චක්ඛු සූත්‍රයේ මෙනි.) 
[2] “මහණෙනි, පඨවිධාතුවෙහි යම් ඡන්දරාගයක් වේද, එය සිතේ කිලුටුවීමයි. ආපෝ ධාතුවෙහි යම් ඡන්දරාගයක් වේද, එය කණෙහි කිලුටුවීමයි. තේජෝ ධාතුවෙහි යම් ඡන්දරාගයක් වේද, එය නාසයෙහි කිලුටු වීමයි. වායෝ ධාතුවෙහි යම් ඡන්දරාගයක් වේද, එය දිවෙහි කිලුටු වීමයි. ආකාශ ධාතුවෙහි යම් ඡන්දරාගයක් වේද, එය කයෙහි කිලුටු වීමයි. විඤ්ඤාණ ධාතුවෙහි යම් ඡන්දරාගයක් වේද, එය සිතේ කිලුටු වීමයි. 

#ඛන්ධ සූත්‍රය
#[1] (මේ සූත්‍රයේ 1, 3 වෙනි ඡෙද කිලෙස සංයුත්තයේ 1 චක්ඛු සූත්‍රයේ මෙනි.) 
[2] “මහණෙනි, රූපස්කන්ධයෙහි යම් ඡන්දරාගයක් වේද, එය සිතේ කිලුටු වීමයි. වේදනාවෙහි යම් ඡන්දරාගයක් වේද, එය කණෙහි කිලුටු වීමයි. සඤ්ඤාවෙහි යම් ඡන්දරාගයක් වේද, එය නාසයෙහි කිලුටු වීමයි. සංස්කාරයන්හි යම් ඡන්දරාගයක් වේද, එය දිවෙහි කිලුටු වීමයි. විඤ්ඤාණයෙහි යම් ඡන්දරාගයක් වේද, එය සිතේ කිලුටුවීමයි. 

#කිලෙස සංයුත්තය නිමි.
#චක්ඛු සූත්‍රය රූප සූත්‍රය, විඤ්ඤාණ සූත්‍රය, ඵස්ස සූත්‍රය, වේදනා සූත්‍රය සඤ්ඤා සූත්‍රය, චේතනා සූත්‍රය, තණ්හා සූත්‍රය, ධාතු සූත්‍රය සහ ඛන්ධ සූත්‍රය දැයි සූත්‍ර දසයකි.
