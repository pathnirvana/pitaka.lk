# සීති වර්‍ගය 
# අනුත්තර සීතිභාව සූත්‍රය 
# ´´මහණෙනි, කරුණු සයකින් යුක්තවූ භික්‍ෂුව නිරුත්තරවූ (එයට වඩා උසස් යමක් නැති) සංසිඳීම් ප්‍රත්‍යක්‍ෂ කරන්නට නුසුදුසුය. 
# ´´කවර සයකින්ද? මහණෙනි, මේ සස්නෙහි භික්‍ෂුව යම් කලෙක සිතට නිග්‍රහ කටයුතු නම් ඒ කාලයෙහි සිතට නිග්‍රහ නොකෙරේද, යම් කලෙක සිත උත්සාහවත් කටයුතු නම්, ඒ කාලයෙහි සිත උත්සාහවත් නොකෙරේද, යම් කලෙක සිත උසස් කටයුතු නම්, ඒ කාලයෙහි සිත උසස් නොකෙරේද, යම් කලෙක සිත මධ්‍යස්ථ කටයුතු නම්, ඒ කාලයෙහි සිත මධ්‍යස්ථ නොකෙරේද, පහත් තත්ත්‍වයෙහි ඇලීම වේද, ආත්ම දෘෂ්ටියෙහි ඇලුනේ වේද, මහණෙනි, කරුණු සයකින් යුක්තවූ භික්‍ෂුව නිරුත්තරවූ (එයට වඩා උසස් යමක් නැති) සංසිඳීම් ප්‍රත්‍යක්‍ෂ කරන්නට නුසුදුසුය. 
# ´´මහණෙනි, කරුණු සයකින් යුක්තවූ භික්‍ෂුව නිරුත්තරවූ සංසිඳීම ප්‍රත්‍යක්‍ෂ කරන්නට සුදුසුය. 
# ´´කවර සයකින්ද? මහණෙනි, මේ සස්නෙහි භික්‍ෂුව යම් කලෙක සිතට නිග්‍රහ කටයුතු නම් ඒ කාලයෙහි සිතට නිග්‍රහ කෙරේද, යම් කලෙක සිත උත්සාහවත් කටයුතු නම්, ඒ කාලයෙහි සිත උත්සාහවත් කෙරේද, යම් කලෙක සිත උසස් කටයුතු නම්, ඒ කාලයෙහි සිත උසස් කෙරේද, යම් කලෙක සිත මධ්‍යස්ථ කටයුතු නම්, ඒ කාලයෙහි සිත මධ්‍යස්ථ කෙරේද, ප්‍රණීත දෙයෙහි ඇළුනේ වේද, නිර්‍වාණයෙහි ඇළුනේද, මහණෙනි, කරුණු සයකින් යුක්තවූ භික්‍ෂුව නිරුත්තරවූ සංසිඳීම ප්‍රත්‍යක්‍ෂ කරන්නට සුදුසුය. 
# නියාමොක්කමන සූත්‍රය 
# ´´මහණෙනි, කරුණු සයකින් යුක්තවූ තැනැත්තේ, බණ අසන්නේද, කුශල ධර්මයන්හි තැන්පත්වූ නියම කළ දෙයට බැසගන්නට නොහැක්කේ වෙයි. 
# ´´කවර සයකින්ද? කර්‍මාවරණ බැවින් යුක්තවූයේ වේද, ක්ලේශාවරණ බැවින් යුක්තවූයේ වේද, අකුශල විපාකයකින් වැසුනේ වේද. බලවත් කැමැත්ත නැත්තේ වේද, ප්‍රඥාවත් නොවේද, මහණෙනි, කරුණු සයකින් යුක්තවූ තැනැත්තේ, බණ අසන්නේද, කුශල ධර්මයන්හි තැන්පත්වූ නියම කළ දෙයට බැසගන්නට නොහැක්කේ වෙයි. 
# ´´මහණෙනි, කරුණු සයකින් යුක්තවූ තැනැත්තේ, බණ අසන්නේ, කුශල ධර්මයන්හි තැන්පත්වූ නියම කළ දෙයට බැසගන්නට හැක්කේ වෙයි. 
# ´´කවර සයකින්ද? කර්‍මාවරණ බැවින් යුක්ත නොවූයේ වේද, ක්ලේශාවරණ බැවින් යුක්ත නොවූයේ වේද, අකුශල විපාකයකින් නොවැසුනේ වේද. බලවත් කැමැත්ත ඇත්තේ වේද, ප්‍රඥාවත් වේද, මහණෙනි, කරුණු සයකින් යුක්තවූ තැනැත්තේ, බණ අසන්නේ, කුශල ධර්මයන්හි තැන්පත්වූ නියම කළ දෙයට බැසගන්නට හැක්කේ වෙයි. 
# සම්මත්ත සූත්‍රය 
# ´´මහණෙනි, කරුණු සයකින් යුක්තවූ තැනැත්තේ, බණ අසන්නේ නමුදු කුශල ධර්මයන්හි තැන්පත්වූ නියම කළ දෙයට බැසගන්නට නොහැක්කේ වෙයි. 
# ´´කවර සයකින්ද? මව මරණ ලද්දේ වේද, පියා මරණ ලද්දේ වේද, රහතුන් වහන්සේ මරන ලද්දේ වේද, දුෂ්ටවූ සිතින් තථාගතයන් වහන්සේගේ ශරීරයෙන් ලේ සොල්වන ලද්දේ වේද, සංඝ භේදය කරණ ලද්දේ වේද, ප්‍රඥා නැත්තේ වේද. මෝඩවූ. කෙළතොලු කෙනෙක් වේද, යන මේ කරුණු සයින් යුක්තවූ තැනැත්තේ මහණෙනි, කරුණු සයකින් යුක්තවූ තැනැත්තේ, බණ අසන්නේ නමුදු කුශල ධර්මයන්හි තැන්පත්වූ නියම කළ දෙයට බැසගන්නට නොහැක්කේ වෙයි. 
# ´´මහණෙනි, කරුණු සයකින් යුක්තවූ තැනැත්තේ, බණ අසන්නේ නමුදු කුශල ධර්මයන්හි තැන්පත්වූ නියම කළ දෙයට බැසගන්නට හැක්කේ වෙයි. 
# ´´කවර සයකින්ද? මව නොමරණ ලද්දේ වේද, පියා නොමරණ ලද්දේ වේද, රහතන් වහන්සේ නොමරණ ලද්දේ වේද, දුෂ්ටවූ සිතින් තථාගතයන් වහන්සේගේ ශරීරයෙන් ලේ නොසොල්වන ලද්දේ වේද, සංඝ භේදය නොකරණ ලද්දේ වේද, ප්‍රඥා ඇත්තේ වේද. මෝඩවූ. කෙළතොලු කෙනෙක් නොවූයේද, මහණෙනි, කරුණු සයකින් යුක්තවූ තැනැත්තේ, බණ අසන්නේ නමුදු කුශල ධර්මයන්හි තැන්පත්වූ නියම කළ දෙයට බැසගන්නට හැක්කේ වෙයි. 
# සුස්සූසන සූත්‍රය 
# ´´මහණෙනි, ධර්‍ම සයකින් යුක්තවූ තැනැත්තේ බණ අසන්නේ නමුදු කුශල ධර්මයන්හි තැන්පත්වූ නියම කළ දෙයට බැසගන්නට නොහැක්කේ වෙයි. 
# ´´කවර සයකින්ද? තථාගතයන් වහන්සේ විසින් දන්නා ලද ධර්‍ම විනය දේශනා කරණ කල්හි නොඅසාද. කණ නොනමාද. වෙන කරුණක් සිතට නගා ගනීද, අනර්ත්‍ථයක්ම ගනීද, අර්ත්‍ථය නොසලකා හරීද. පිළිවෙලින් තොරවූ ඉවසීමෙන් යුක්ත වේද. මහණෙනි, ධර්‍ම සයකින් යුක්තවූ තැනැත්තේ බණ අසන්නේ නමුදු කුශල ධර්මයන්හි තැන්පත්වූ නියම කළ දෙයට බැසගන්නට නොහැක්කේ වෙයි. 
# ´´ මහණෙනි, ධර්‍ම සයකින් යුක්තවූ තැනැත්තේ බණ අසන්නේ නමුදු කුශල ධර්මයන්හි තැන්පත්වූ නියම කළ දෙයට බැසගන්නට හැක්කේ වෙයි. 
# ´´කවර සයකින්ද? තථාගතයන් වහන්සේ විසින් දන්නා ලද ධර්‍ම විනය දේශනා කරණ කල්හි අසාද. කන් නමාද. වෙන කරුණක් සිතට නගා නොගනීද, අර්ත්‍ථයක්ම ගනීද, අර්ත්‍ථය සලකාද. පිළිවෙලින් යුක්තවූ ඉවසීමෙන් යුක්ත වේද. මහණෙනි, ධර්‍ම සයකින් යුක්තවූ තැනැත්තේ බණ අසන්නේ නමුදු කුශල ධර්මයන්හි තැන්පත්වූ නියම කළ දෙයට බැසගන්නට හැක්කේ වෙයි. 
# දිඨිසම්පදා සච්ඡීකරණ සූත්‍රය 
# ´´මහණෙනි, කරුණු සයක් දුරු නොකොට සෝවාන් බව ලැබීමට නුසුදුසුය. 
# ´´කවර සයක්ද? ආත්ම දෘෂ්ටිය, සැකය, අන්‍ය ශීල ව්‍රතයන් යහපත්යයි ගැනීම, අපායට යාමට හේතුවන රාගය, අපායට යාමට හේතුවන ද්වේෂය, අපායට යාමට හේතුවන මෝහය, මහණෙනි, කරුණු සයක් දුරු නොකොට සෝවාන් බව ලැබීමට නුසුදුසුය. 
# ´´ මහණෙනි, කරුණු සයක් දුරු කොට සෝවාන් බව ලැබීමට සුදුසුය. 
# ´´කවර සයක්ද? ආත්ම දෘෂ්ටිය, සැකය, අන්‍ය ශීල ව්‍රතයන් යහපත්යයි ගැනීම, අපායට යාමට හේතුවන රාගය, අපායට යාමට හේතුවන ද්වේෂය, අපායට යාමට හේතුවන මෝහය, මහණෙනි, කරුණු සයක් දුරු කොට සෝවාන් බව ලැබීමට සුදුසුය. 
# පහීන ධම්මඡක්ක සූත්‍රය 
# ´´මහණෙනි, කරුණු සයක් සෝවාන් පුද්ගලයාට ප්‍රහීණ වූවාහු වෙති. 
# ´´කවර සයක්ද? ආත්ම දෘෂ්ටිය, සැකය, අන්‍ය ශීල ව්‍රතයන් යහපත්යයි ගැනීම, අපායට යාමට හේතුවන රාගය, අපායට යාමට හේතුවන ද්වේෂය, අපායට යාමට හේතුවන මෝහය, මහණෙනි, කරුණු සයක් සෝවාන් පුද්ගලයාට ප්‍රහීණ වූවාහු වෙති. 
# උප්පාදනායාභබ්බධම්ම සූත්‍රය 
# ´´මහණෙනි, සෝවාන් පුද්ගලයා කරුණු සයක් උපදවන්නට නුසුදුසු වෙයි. 
# ´´කවර සයක්ද? ආත්ම දෘෂ්ටිය, සැකය, අන්‍ය ශීල ව්‍රතයන් යහපත්යයි ගැනීම, අපායට යාමට හේතුවන රාගය, අපායට යාමට හේතුවන ද්වේෂය, අපායට යාමට හේතුවන මෝහය යන මේ සය වෙති, මහණෙනි, සෝවාන් පුද්ගලයා කරුණු සයක් උපදවන්නට නුසුදුසු වෙයි. 
# ඡඅභබ්බට්ඨාන සූත්‍රය 
# ´´මහණෙනි, මේ නුසුදුසු කරුණු සයක් වෙති. 
# ´´කවර සයක්ද? සෝවාන් පුද්ගලයා ශාස්තෲන් වහන්සේ කෙරෙහි ගෞරව නැතිව, නොකීකරුව වාසය කරන්නට නුසුදුසුය. සෝවාන් පුද්ගලයා ධර්‍මය කෙරෙහි ගෞරව නැතිව, නොකීකරුව වාසය කරන්නට නුසුදුසුය. සෝවාන් පුද්ගලයා සංඝයා කෙරෙහි ගෞරව නැතිව, නොකීකරුව වාසය කරන්නට නුසුදුසුය. සෝවාන් පුද්ගලයා හික්‍ෂාව කෙරෙහි ගෞරව නැතිව, නොකීකරුව වාසය කරන්නට නුසුදුසුය. සෝවාන් පුද්ගලයා සංඝයා කෙරෙහි ගෞරව නැතිව, වාසය කරන්ට නුසුදුසුය. සෝවාන් පුද්ගලයා ශික්‍ෂාව කෙරෙහි ගෞරව නැතිව, නොකීකරුව වාසය කරන්නට නුසුදුසුය. සෝවාන් පුද්ගලයා නොපැමිණෙන ස්ථානයට හැරී ඒමට නුසුදුසුය. සෝවාන් පුද්ගලයා අටවෙනි භවයක් උපදවන්නට නුසුදුසුය. මහණෙනි, මේ නුසුදුසු කරුණු සයක් වෙති. 
# කිඤ්චිසංඛාරූප ගමන සූත්‍රය 
# ´´ මහණෙනි, මේ නුසුදුසු කරුණු සයක් වෙති. 
# ´´කවර සයක්ද? සෝවාන් පුද්ගලයා කිසි සංස්කාරයක් නිත්‍ය වශයෙන් ගන්නට නුසුදුසු වෙයි. සෝවාන් පුද්ගලයා කිසි සංස්කාරයක් සැප වශයෙන් ගන්නට නුසුදුසුය. සෝවාන් පුද්ගලයා කිසි ධර්‍මයක් ආත්ම වශයෙන් ගන්නට නුසුදුසුය. සෝවාන් පුද්ගලයා ඇසූ දුටු දේ මගුල් වශයෙන් පිරිසිදුයයි ගෙන හැරීයාමට නුසුදුසුය. සෝවාන් පුද්ගලයා මේ ශාසනයෙන් පිටත දන් දීමට සුදුස්සන් සෙවීමට නුසුදුසුය. මහණෙනි, මේ නුසුදුසු කරුණු සයක් වෙති. 
# ආනන්තරිය සූත්‍රය 
# ´´මහණෙනි, මේ නුසුදුසු කරුණු සයක් වෙති. 
# ´´කවර සයක්ද? සෝවාන් පුද්ගලයා මව මැරීමට නුසුදුසුය. පියා මැරීමට නුසුදුසුය. රහතන් වහන්සේ මැරීමට නුසුදුසුය. දුෂ්ටවූ සිතින් තථාගතයන් වහන්සේගේ ශරීරයෙන් ලේ සෙලවීමට නුසුදුසුය. සංඝභේදය කරන්නට නුසුදුසුය. සෝවාන් පුද්ගලයා අන්‍ය ශාස්තාාවරයකු ඇතැයි පෙන්විමට නුසුදුසුය.´´ 
# සයංකත සුඛදුක්ඛ සූත්‍රය 
# ´´මහණෙනි, නුසුදුසු කරුණු සයක් වෙති. කවර සයක්ද? 
# ´´මහණෙනි. සෝවාන් පුද්ගලයා තමා විසින් කරණ ලද සුවදුක් වලට ආපසු හැරී ඒමට නුසුදුසුය. අනුන් විසින් කරණ ලද සැපදුක් වලට ආපසු හැරී ඒමට නුසුදුසුය. තමා විසින්ද, අනුන් විසින්ද කළ සැප දුක්වලට ආපසු හැරී ඒමට නුසුදුසුය. තමා විසින් නොකරණ ලද හේතුවක් නැතිව හටගත් සැප දුක් වලට හැරී ඒමට නුසුදුසුය. 
# ´´සෝවාන් පුද්ගලයා තමා විසින්ද. අනුන් විසින්ද නොකරණ ලද හේතුවක් නැතිව හටගත් සුවදුක්වලට ආපසු හැරී ඒමට නුසුදුසුය. 
# ´´එයට හේතු කවරේද? මහණෙනි, එසේමය. සෝවාන් පුද්ගලයා විසින් හේතුව දක්නා ලදී. හේතුවෙන් යුක්ත ධර්මයෝද දක්නා ලදී. මහණෙනි, නුසුදුසු කරුණු සයක් වෙති. කවර සයක්ද?
