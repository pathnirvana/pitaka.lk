# ´´මම සංක්‍ෂිප්ත ලෙස විචාරණ කල්හිදු විස්තර වශයෙන් කියමි. (විචාරන්නාවූ) ඔවුන්ගේ අදහස් සම්පූර්ණ කරමි. විපදෝත්තමවූ බුදුරජාණන් වහන්සේ සතුටු කරමි.
# ´´මා විසින් තුටු කරණ ලද්දාවූ, මහා වීරවූ, සියල්ල තෙමේම දක්නාවූ, අග්‍ර පුද්ගලවූ (ජ්‍යෙෂ්ඨ බුදුරජ තෙම) බික් සඟන මැද හිඳ මා එතදග්‍රස්ථානයෙහි තැබීය.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් මහා කච්චායන තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ __________ 
# කාළුදායී ථෙරාපදානය.
#
# ´´මම එකල්හි ලොක ජ්‍යෙෂ්ඨවූ, තාදී ගුණ ඇති, මහමගට පිළිපන්, චාරිකාවෙහි හැසිරෙන්නාවූ, ශාස්තෘවූ, පදුමුත්තර බුදුරජාණන් වහන්සේට මැනවින් පිපුණු පියුම්ද, උපුල්ද, බෝලිද්ද ආහාර (දානය) ද ගෙන (එහි) ගියෙමි.
# ´´මහාවීර තෙම උතුම් ආහාරය වූ මනා භෝජනය වළඳයි. ඒ මල්ද ගෙන මහජනයාට දැක්වීය. (කෙසේද යත්?)
# ´´(මේවා) ලෝකයෙහි ඉෂ්ටවූ, මනවඩන්නාවූ, ප්‍රියවූ උතුම් ජලජ (ජලයෙහි හටගන්නාවූ) පුෂ්පයෝ වෙති. යමෙක් තෙම මට මේ මල් දුන්නේද, ඔහු විසින් ඉතා දුෂ්කර පින්කමක් කරණ ලදී.
# ´´යමෙක් මට මල් නගා දුන්නේද, උතුම්වූ ආහාරය දුන්නේද, ඔහු (කවරෙක්දැයි) මම කියන්නෙමි. කියන්නාවූ මාගේ වචනය අසව් (කෙසේදයත්?)
#
# ´´හෙතෙම අටලොස් වරක් දිව්‍ය රාජ්‍යය කරන්නේය. එකෙණෙහි ඔහුගේ පුණ්‍ය විපාකයෙන් උපුල්ද, පියුම්ද, බෝලිද්දද, එයටද වැඩි දිව්‍යමය සුගන්‍ධයෙන් යුතුව අහසෙහි වියනක් කොට දරන්නේය.
# ´´පස්විසි වරක් සක්විති රජ වන්නේය. පන්සිය වරක් පෘථිවි රාජ්‍යය ලබා පොළොවෙහි වසන්නේය.
# ´´(මින්) කල්ප ලක්‍ෂයකින් මතුයෙහි ඔක්කාක කුලයෙහි උපදින්නාවූ, නම්න් ගෞතම නම් වූ, ශාස්තෘතෙම ලොව පහළවන්නේය.
# ´´කුශල මූලයෙන් මෙහෙයන ලද ස්වකිය කටයුතු පටන්ගත් හෙතෙම ශාක්‍යයින්ගේ සතුට ඇතිකරන්නාවූ නෑ සබඳකම් ඇත්තෙක් වන්නේය.
# ´´කුශලමූලයෙන් මෙහෙයන ලද හෙතෙම පසුව පැවිදිව සියලු ආශ්‍රවයන් ප්‍රහාණ පරිඥා වශයෙන් දැන (දුරුකොට) ආශ්‍රව රහිතව පිරිනිවන් පාන්නේය.
# ´´ලොක බන්ධුවූ ගෞතම බුදුරජාණන් වහන්සේ පැමිණි (සිව්) පිළිසිඹියාවන් ඇතිකරණලද කටයුතු ඇති ආශ්‍රව රහිතවූ හේ අග්‍ර පදවියෙහි ගබන්නේය. (කෙසේද යත්?)
# ´´වීර්‍ය්‍ය වැඩූ, නිවණ කෙරහි මෙහෙයූ සිත් ඇති, උපශාන්තවූ, උපධි රහිතවූ, හෙතෙම නම්න් උදායී නම් වූ බුද්‍ධශ්‍රාවකයෙක් වන්නේය කියායි.
# ´´රාගයද, ද්වේෂයද, මොහයද, මක්ඛය (ගුණමතු බව) ද, විනාශ කරණ ලදුයේ, සියලු ආශ්‍රවයන් ප්‍රහාණ පරිඥාවෙන් දැන (දුරුකොට) ආශ්‍රව රහිතව වෙසෙමි.
# ´´කෙලෙස් තවන වීර්‍ය්‍යයෙන් යුක්තවූ, ප්‍රඥාවත්වූ, මම බුදුරජාණන් වහන්සේ සතුටු කෙළෙමි. ප්‍රමුදිතවූ බුදුරජ තෙම මා අග්‍ර පදවියෙහි තැබීය.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් කාළුදායී තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ __________
# මොඝරාජ ථෙරාපදානය.
# ´´සියල්ල තෙමේම දැනගත්තාවූ, අපරාජාතවූ, (පස්මරුන් දිනූ) අත්‍ථදස්සී භාග්‍යවතුන් වහන්සේ භික්‍ෂු සංඝයා විසින් පිරිවරණ ලදුයේ වීථියට පිළිපන් සේක.

# ´´මම (ළුහුණු) කදක් පුරවාගෙන සඞ්ඝාරාමයට ගියෙමි. සතුටුවූයේ සතුටු සිතින්ම සංඝයාට ළුහුණු දුනිමි.
# ´´මම විපස්සි නරශ්‍රෙෂ්ඨයන් (බුදුරජාණන්) වහන්සේගේ සසුනෙහි නිරතවු සංඝයාට ළුහුණු දී කල්පයක් මුළුල්ලෙහි ස්වර්‍ගයෙහි සතුටුවීමි.
# ´´(මින්) එකානූවන කල්පයෙහිවූ එකල්හි (මම) යම් ළුහුණු දනක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. ළුහුණු දානයාගේ මේ විපාකයි.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් ලසුනදායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ __________ 
# ආයාගදායක ථෙරාපදානය.
# ´´කථා කරන්නවුන් අතුරෙහි උතුම්වූ සිඛී බුදුරජාණන් වහන්සේ පිරිනිවි කල්හි (මම) සතුටුවූයේ, සතුටු සිතින් ශ්‍රේෂ්ඨ චෛත්‍යය වැන්දෙමි.
# ´´මම එකල්හි වඩුවන් සමග කථාකොට මුදල් දී සතුටුවූයේ, සතුටු සිතින් දීර්ඝ භෝජන ශාලාවක් කරවීමි.
# ´´(ඒ හේතුවෙන්) මම අට කපක් නොකඩකොට දෙව්ලොව විසීමි. සෙසු කල්පයන්හි (දිව්‍ය මනුෂ්‍ය දෙගතියෙහි) මිශ්‍ර ලෙස සැරි සැරීමි.
# ´´මාගේ කයෙහි විෂ ඇතුළු නොවේ. ආයුධ මා නොනසත්. දියෙහි මම නොමැරෙමි. ආයාග (දීර්‍ඝ ශාලා) දානයාගේ මේ විපාකයයි.
# ´´ඉදින් මම වැස්ස කැමතිවෙම් නම් මහ වැසි වසින්නේය. දෙවියෝද මාගේ වසඟයට පැමිණෙත්. (ආයාග දානමය) පුණ්‍ය කර්‍මයාගේ මේ විපාකයයි.
# ´´මම තිස් වරක් සත් රුවනින් යුක්තවූ සක්විති රජ වීමි. මට කිසිවෙක් නින්දා නොකෙරෙත්. පුණ්‍ය කර්මයාගේ මේ විපාකයයි.
# මින් තිස් එක්වන කල්පයෙහි (මම) යම් දීර්ඝ ශාලාවක් කරවීම්ද (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. දීර්ඝ ශාලා දානයාගේ මේ විපාකයයි.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් ආයාගදායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ __________ 
# ධම්මචක්කික ථෙරාපදානය.
# ´´සිද්‍ධාර්ත්‍ථ භාග්‍යවතුන් වහන්සේගේ ධර්‍මාසනය ඉදිරියෙහි මා විසින් මැනවින් කරණ ලද, නුවණැත්තවුන් විසින් වර්‍ණනා කරණ ලද ධර්ම චක්‍රයක් තබන ලදී.
# ´´මනොඥවූ (ස්වර්‍ණ) වර්‍ණවූයේ (ස්වර්‍ණ සිවිකාවි) යානද (සේනාපති - මහාමාත්‍යාදී) සේවා භටයන්ද (හස්ති - අශ්ව - රථ නම්) වාහනද ඇත්තේ බබළමි. මා අනුව පවතින්නාවූ බොහෝ දෙනා නිරතුරුවම මා පිරිවරත්.
# ´´මම හැමකල්හි සැටදහසක් තූර්‍ය්‍යයන්ගෙන් පරිවරණය කරමි. පිරිවර කරණකොටගෙන බබළමි. පුණ්‍ය කර්‍මයාගේ මේ පලයයි.
# ´´(මින්) සිව් අනූවන කල්පයෙහි මම යම් (ධර්‍ම) චක්‍රයක් තැබූයෙම්ද (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. ධර්‍මචක්‍ර දානයාගේ මේ විපාකයයි.
# ´´මින් එකොළොස්වන කල්පයෙහි නම්න් සහස්සරාජ නම්වූ, ජනාධිපතිවූ, මහත් බල ඇති සක්විති රජවරු අටදෙනෙක් (අට වරකට) වූවාහුය.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් ධම්මචක්කික තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ __________ 
# කප්පරුක්ඛිය ථෙරාපදානය.
# ´´මම සිද්‍ධාර්ත්‍ථ බුදුරජාණන් වහන්සේගේ ශ්‍රේෂ්ඨ ස්ථූපය ඉදිරියෙහි විසිතුරු දුහුල් එල්වා කප් රුකක් තැබීමි.
# ´´දිව්‍යවූ හෝ, මනුෂ්‍යවූ හෝ, යම් යම් යෝනියක උපදිම්ද, (ඒ හැම අත්බැව්හිදීම) මාගේ දොරටුව හොබවමින් කප් රුකක් එහි පිහිටයි.
# ´´මමද, මා ඇසුරු කළ යම් කිසි පිරිසක් වෙත් නම් ඔවුහුද යන අපි හැම කල්හි එම කප් රුකින් දුහුල් ගෙන හැඳීමු.
